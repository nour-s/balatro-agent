# Egg

at end of round Copyable Perishable Eternal

Egg is a Common Economy Joker which gains $3 of sell value each round, and has no other effect. It can help with some joker synergies, or simply be cashed in after a few rounds for a small profit.

## Usage Notes

- The Omelette ( All give no reward money<br>Extra no longer earn money<br>Earn no at end of round 5x Banned Vouchers:<br><br><br><br>Banned Jokers:<br><br><br><br>) challenge starts the player with 5 copies of Egg.

## Synergies

### Decks

The Green Deck ( At end of each Round: <br> per remaining <br> per remaining <br> Earn no Discover at least items from your [[collection]] ) cannot generate interest, and can especially benefit from alternative income sources like Egg.

### Poker Hands

This Joker does not add Chips or Mult, so no hand benefits or suffers from it.

### Jokers

- Swashbuckler (Adds the sell value of all other owned [[Jokers|]] to Mult <br> }} ) can combo with Egg to gain Mult each round, but will lose it if Egg is sold. While Debuffed, Egg will still provide Mult for Swashbuckler.

- Ceremonial Dagger (When [[Blinds|]] is selected, destroy [[Jokers|Joker]] to the right and permanently add its sell value to this <br> }} ) can be used to sacrifice Egg for a permanent Mult bonus.

### Tarot Cards

- The money given by Temperance (Gives the total sell value of all current [[Jokers]]<br> }} Available from start. ) scales with the sell value of the player's Jokers. The longer players have Egg, the more money they'll get from Temperance, even potentially hitting the maximum value of $50 they can get from the Tarot card.

### Vouchers

- Hieroglyph ( Ante, <br> ) and Petroglyph ( Ante, <br> ) send the player back an Ante, giving players more time to scale Egg's sell value.

## Anti-Synergies

### Decks

- Anaglyph Deck ( After defeating each [[Blinds|]], gain a [[File:|24px]] [[Tags|]] Win a run with any [[Decks|deck]] ) incentives skipping blinds due to its Double Tag ( ) gimmick, which gives Egg less chances to increase its sell value.

- Since Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ) and Blast Off ( hands per round<br> discards per round<br> Joker Slots Eternal <br>Eternal <br><br>Banned [[Vouchers]]: , <br>Banned Jokers: ) only have 4 Joker slots instead of the typical 5, taking Egg is much more risky since it doesn't immediately help the player score and its only purpose is to be sold.

- Cruelty ( give no reward money<br> give no reward money<br><br> Joker Slots ) makes it even more difficult and risky to use Egg, as this deck only has 3 Joker slots.

- Non-Perishable ( All Jokers are Eternal Banned Jokers: , ; , , ; ; , , and <br>Banned [[Blinds]]: ) makes every Joker Eternal (), including Egg, which means it cannot be sold.

### Jokers

- Throwback ( for each [[Blinds|]] skipped this run <br> }} ) encourages skipping blinds to scale the XMult of the Joker, once again giving Egg less opportunity to increase its sell value.

### Stakes

- On Black Stake and above, there is a chance for an Egg to be Eternal (), which makes it impossible to sell if picked up, nullifying the main point of the Joker.

- On Gold Stake, an Egg can be Rental (), taking away $3 from the player every round, which is usually not worth the sell value increase.

## Trivia

- LocalThunk has confirmed1] Egg is a reference to:

 - Northernlion, a Twitch streamer and YouTuber who has played Balatro.

 - The term "nest egg", which is a reserve or investment of money for the future, fitting for the Joker's gimmick of its sell value gradually increasing each round.

- It may also be referencing the Clowns International Egg Registry, where clowns can register their unique face makeup by painting it on an egg.

- Egg is tied with DNA (If of round has only card, ) for the shortest Joker name in Balatro, containing only 3 letters.

- This Joker is part of the group of "Consumable" Jokers that can be destroyed with usage, dubbed "Food Jokers" due to their shared theme. The other Jokers in this group are Ice Cream (<br> ), Popcorn ( <br> ), Gros Michel (<br> ), Cavendish ( <br> ), Seltzer (), Turtle Bean ( hand size, reduces by each round Uncommon Effect N/A (Passive) ), Ramen (, ), and Diet Cola (Sell this card to create a free [[Tags|]] Uncommon Effect N/A (Passive) ).2]3]4]

- Egg is one of the few "food jokers" to get merch from Makeship in the recently campaigned Food Joker Pin Set alongside Popcorn ( <br> ) and Ice Cream (<br> ).5]
