# Fortune Teller

card used this run
 (Currently +0) Copyable Perishable Eternal

Fortune Teller is a Common scaling Additive Mult Joker that gives +1 Mult per Tarot Card used. It starts at +0 Mult. The effect is retroactive, meaning players immediately benefit from all the Tarot cards they utilized before buying this Joker.

## Synergies

### Decks

- Zodiac Deck ( Start run with , , and Win a run with any [[Decks|deck]] ), with its Tarot Merchant ( ) already equipped, benefits greatly from Fortune Teller, as players have more chances to buy and use Tarot Cards without having to rely on Booster Packs.

- Yellow Deck ( Start with extra Discover at least items from your [[Collection|collection]] ) extra $10 may allow for more Tarot Cards to be bought.

### Poker Hands

Any Poker hand can benefit from Fortune Teller.

### Jokers

- Vagabond (Create a [[Tarot Cards|]] ), Cartomancer (Create a [[Tarot Cards|]] card when [[Blinds|]] is selected <br> Discover every Tarot Card. ), Hallucination ( chance to create a [[Tarot Cards|]] card when any [[Booster Packs|]] is opened <br> Common Effect N/A (Passive) ), 8 Ball ( chance for each played to create a [[Tarot Cards|]] card when scored<br> ), and Superposition (Create a [[Tarot Cards|]] card if poker hand contains an and a <br> ) can all provide Tarot cards with which to nourish Fortune Teller.

- Chaos the Clown ( free per shop Common Effect N/A (Passive) ) gives a free reroll in the Shop allowing for more Tarot Cards to be found.

- Certificate (When round begins, add a random with a random [[Seal|]] to your hand Have a [[Gold_Cards|Gold playing card]] with a [[Gold_Seal|]] Uncommon Effect N/A (Passive) ) can generate cards with a Purple Seal (Creates a card when <br> ), granting a repeatable source of Tarot Cards.

- Vampire (This Joker gains per scoring played, removes card <br> }} ) consumes Enhancements from played cards, making it a good combo with Fortune Teller.

### Vouchers

- Overstock ( ) and Overstock Plus ( ) respectively add a 3rd and 4th card slot in the Shop, giving the player more chances to find Tarot Cards to buy and use.

- Reroll Surplus () and Reroll Glut () make it cheaper to reroll to find more Tarot cards to buy and use.

- Clearance Sale () and Liquidation () reduce the price of Tarot Cards and Arcana Packs.

- Tarot Merchant ( ) and Tarot Tycoon ( ) make it more likely to find Tarot Cards in the shop.

 - Crystal Ball ( ) gives +1 consumable slot, so you can purchase more Tarot Cards in the shop for the next round.

### Modifiers

- Purple Seal (Creates a card when <br> ) can generate Tarot Cards when discarding, allowing Fortune Teller to scale faster.

### Consumables

Of course, every Tarot Card benefits Fortune Teller, but there are 2 in particular that benefit the most.

- The Emperor (Creates up to random cards<br> Available from start. ) provides two Tarot Cards.

- The Fool (Creates the last [[Tarot Cards|]] or [[Planet Cards|]] card used during this run<br><small> excluded</small> Available from start. ) can copy a Tarot Card to use.

- Medium (Add a [[Purple Seal|]] to selected card in your hand Available from start. ) creates a Purple Seal (Creates a card when <br> ), allowing the player to create more Tarot Cards.

### Other Synergies

It's beneficial to use as many Tarot cards as possible to buff up Fortune Teller. Even if the Tarot cards provide no benefit (like suit changes, etc.), they can still be used on dummy cards.

## Trivia

- Tarot card readings are occasionally used by fortune tellers to help them make predictions.

- The color of "Tarot" in Fortune Teller's description is incorrectly formatted as {C:purple} instead of {C:tarot}, which is only used correctly in Medium (Add a [[Purple Seal|]] to selected card in your hand Available from start. ). This makes the text appear as a slightly darker purple than other cards that reference Tarot Cards.

 - Vagabond (Create a [[Tarot Cards|]] ) has the same issue.
