# Ancient Joker

when scored,
 suit changes at end of round Copyable Perishable Eternal

Ancient Joker is a Rare Multiplicative Mult Joker with highly variable power. At the end of each round, a new suit is chosen evenly amongst the 3 suits it is not currently using, regardless of the composition of the current deck. If multiple Ancient Jokers are held, they will all pick the same suit as each other. Each time a card of the current suit is scored, Ancient Joker applies X1.5 Mult.

This effect can vary from incredibly powerful to practically irrelevant. Ancient Joker can carry a run into the late Antes, but only if the deck is built specifically to take advantage of its effect.

## Synergies

### Decks

- Black Deck ( Joker slot <br> hand every round Discover at least items from your [[collection]] ) allows for stronger Joker synergies to combine with and build around Ancient Joker and works well to mitigate the fewer hands Black Deck gives.

- Nebula Deck ( Start run with the voucher <br> consumable slot Win a run with the on [[Stakes|any difficulty]] ) can level up Flushes very quickly, making Ancient Joker work off of a higher base Mult as well as providing modest Chip scaling.

- Zodiac Deck ( Start run with , , and Win a run with any [[Decks|deck]] ) allows you to see more Tarot cards, which can create more Wild Cards or other strong enhancements, as well as more Jokers, giving more chances to see one that would work well with Ancient Joker.

- Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ) increases your hand size, making it easier to build flushes of the suit Ancient Joker has selected each round.

### Poker Hands

All Flush hands benefit greatly (though inconsistently) from Ancient Joker, since they will trigger its effect the maximum number of times compared to other hands. Unlike other Flush-based builds, it is best to not centralize around one suit, as Ancient Joker will only select it one in three times (1/4th on the first selection). Wild Cards add flexibility in triggering Ancient Joker when other suits are chosen.

### Jokers

Ancient Joker provides good scoring by itself, but this can be significantly increased with the effects of other Jokers.

- Since Ancient Joker is a "When Scored" effect, retriggering cards will repeat the application of Ancient Joker, such as Dusk (), Sock and Buskin (), Hack (Retrigger each played , , , ), and Seltzer ().

- Similarly, Jokers that apply additive Mult effects when cards are scored, such as Fibonacci (Each played , , , , ), Even Steven (Played cards with [[Card Ranks|]] rank give when scored <br> ), and Walkie Talkie (), can benefit from Ancient Joker. Most notable are other Jokers that trigger X Mult effects, though. Mostly, this will be Photograph (), since the others require heavy centralization around suits and ranks, but they will all scale off the effects of retrigger Jokers.

- The Tribe ( if played hand contains a Win a run without playing a . ) can work especially well, since the player is ideally always playing Flushes, making it a smaller, consistent bonus.

- Ways to level up Flushes are very strong in combination with Ancient Joker. The best ways to do this with Jokers are through Burnt Joker (Upgrade the level of the first poker hand each round Sell 50 cards. ), Space Joker ( chance to upgrade level of played [[Poker Hands|]] ), and Astronomer (All cards and in the shop are Discover every card Uncommon Effect N/A (Passive) ), though other Jokers can help with planet scaling in other ways, like DNA (If of round has only card, ) to duplicate Blue Sealed cards.

- Smeared Joker ( and count as the same suit, and count as the same suit Have at least [[Wild Cards|]] in your deck Uncommon Effect N/A (Passive) ) makes the effect of Ancient Joker much more consistent to trigger, as well as allowing for easier Flush hands of either suit pair.

### Consumables

- The Lovers (Enhances selected card into a [[Card Modifiers|]] Available from start. ) works very well, as it makes Wild Cards, which always trigger Ancient Joker.

- Deja Vu (Add a [[Red Seal|]] to selected card in your hand Available from start. ) can apply a Red Seal () to a card, giving retriggers for Ancient Joker.

- Trance (Add a [[Blue Seal|]] to selected card in your hand Available from start. ) can apply a Blue Seal (Creates the card for final played [[Poker Hands|]] of round if in hand<br> ) to a card, allowing for easy planet scaling.

## Anti-Synergies

### Decks

- Checkered Deck ( ) should be avoided, as it only has two suits, meaning Ancient Joker will be inactive half of the time.

- Erratic Deck ( All [[Card Ranks|]] and [[Suits|]] in deck are randomized Win a run with any [[Decks|deck]] ) will rarely have evenly distributed suits, making it harder to use Ancient Joker.

### Jokers

- While Four Fingers (All and can be made with 4 cards Uncommon Effect N/A (Passive) ) can make Flushes easier to play, it enables the player to do so with fewer overall cards, therefore the scoring power of Ancient Joker is weakened if the player opts for a 4-card Flush.

 - It may be suitable to hunt for Planet cards if Four Fingers must remain in conjunction with Ancient Joker.

- Suit-based Jokers like Rough Gem () or Lusty Joker () will not get a lot of benefits since Ancient Joker changes the suit each round.

- Castle (This Joker gains per discarded [[Card Suits|]] card, suit changes every round <br> }} ) can be awkward to use, especially if they both choose the same suit.

- Additive Mult Jokers that trigger during the Joker scoring step, such as Mystic Summit ( ), cannot benefit from the Multiplicative Mult Ancient Joker gives.

### Blinds

- The Goad ( ), The Head ( ), The Club ( ), and The Window ( ) all debuff a specific suit, rendering Ancient Joker useless if it happens to pick the to-be-debuffed suit upon entering the Boss Blind. One trick to avoid this is to check the suit Ancient Joker chose before entering the Big Blind. If it is not the to-be-debuffed suit, skip the Blind to ensure Ancient Joker can be used during the Boss Blind.

### Consumables

- The Sun (Converts up to selected cards to Available from start. ), The Star (Converts up to selected cards to Available from start. ), The Moon (Converts up to selected cards to Available from start. ), and The World (Converts up to selected cards to Available from start. ) can be awkward to use, as Ancient Joker does not care about suit distribution.

- Sigil (Converts all cards in hand to a single random [[Card Suits|]] Available from start. ) can be awkward to use due to the uneven suit distribution.

## Trivia

- Unlike Castle (This Joker gains per discarded [[Card Suits|]] card, suit changes every round <br> }} ), which chooses its suit proportionally to the composition of the current deck, Ancient Joker chooses evenly amongst the 3 suits it is not currently using, regardless of the current deck's composition.

- The artwork on the card is a reference to OLBAP, a face that Twitch streamer "michaelalfox" (MALF) would often sketch when streaming games that involve drawing. This is confirmed by both Localthunk1] and michaelalfox2].
