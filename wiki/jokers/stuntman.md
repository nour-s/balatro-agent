# Stuntman

hand size Copyable Perishable Eternal

Stuntman is a Rare Chips Joker that gives +250 Chips at the cost of -2 hand size.

While Stuntman may appear to be weak at first, it provides an absolutely incredible amount of chips. Given its downside, Stuntman is one of the largest payoffs for running low-scoring hands such as High Card and Pair.

## Usage Notes

- When obtaining Stuntman within a round, the hand size will decrease, but the current number of cards held remains unchanged. When Stuntman is lost in a round, the hand size will increase while drawing cards immediately. This can be utilized to hold cards far exceeding the hand size (see Strategies).

## Synergies

### Decks

- Plasma Deck ( Balance [[Chips|]] and [[Mult|]] when calculating score for played hand <br> base Blind size Win a run with any [[Decks|deck]] ) works very well with Stuntman, as the +250 Chips, when balanced, guarantee players a significant score each hand. Having two Stuntmen can practically clear antes 1-8 by themselves with the Plasma deck, as long as the player does not find themselves with The Psychic ( ).

- Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ) starts with +2 hand size, thus Stuntman's penalty is completely neutralized. However, the -1 Joker Slot penalty makes it harder to build +Mult.

### Poker Hands

Because Stuntman greatly reduces hand size, smaller hands like High Card, Pair, and Three of a Kind are considered the best option.

### Jokers

- Juggler ( hand size Common Effect N/A (Passive) ): +1 hand size, partially offsetting the -2 hand size penalty.

- Turtle Bean ( hand size, reduces by each round Uncommon Effect N/A (Passive) ): The +5 hand size completely removes Stuntman's penalty but is only temporary.

- Troubadour ( hand size, <br> hand each round Win consecutive rounds by <abbr title="Discards are OK to use.">playing only 1 hand</abbr> Uncommon Effect N/A (Passive) ): +2 hand size, which completely removes Stuntman's penalty, at the cost of -1 hand per round.

- Blueprint (Copies ability of [[Jokers|]] ) and Brainstorm (Copies the ability of leftmost [[Jokers|]] Discard a . ) can both copy Stuntman's 250 Chips bonus without the -2 hand size penalty.

- Blackboard ( ): A smaller hand size makes it easier to trigger Blackboard's condition.

- Blue Joker ( for each remaining card in <br> }} ) will give slightly more chips in addition to the ones provided by Stuntman.

- Raised Fist (): The reduced hand size makes it easier to play all the smaller ranked cards from your hand. An early Stuntman + Raised Fist can score over 5,000 for each high card played, making it arguably the most efficient money-to-score ratio that two jokers have together.

- Green Joker ( per hand played <br> per discard <br> }} ): Stuntman wants you to play easier to make hands, such as High Card or Pair, which you don't need to discard for, making Green Joker's downside harder to trigger.

### Consumables

- If Stuntman is generated through a Judgement (Creates a random [[Jokers|]] card<br> Available from start. ) or Wraith (Creates a random [[Rare Jokers| ]], sets money to Available from start. ), the hand size will only drop during the next card draw.

### Vouchers

- Paint Brush ( ) and Palette ( hand size Reduce your hand size down to five cards. ) vouchers both increase hand size by one, counteracting Stuntman's hand size debuff.

### Blinds

- When playing against Crimson Heart ( ), any Joker that gets disabled other than Stuntman will return hand size to normal, but the current hand will stay intact until the next discard (ex 8/6). By quitting to the Main Menu and returning, the hand size will actually permanently increase (ex 8/8), and Stuntman will keep gaining hand size every time another Joker gets disabled.

## Anti-Synergies

### Decks

- Five-Card Draw ( discards per round<br> hand size<br> Joker slots <br>Banned Jokers: , , )'s reduced hand size may make it impossible to play certain poker hands.

### Jokers

- Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) )'s additional -1 hand size penalty may make it impossible to play certain poker hands.

- Jokers with held-in-hand abilities vary with Stuntman's effectiveness.

 - Baron (), Shoot the Moon (Each held in hand gives Play every Heart card in your deck in one round. ), and Reserved Parking () are all weakened due to holding fewer face cards in hand.

### Consumables

- Ouija (Converts all cards in hand to a single random [[Card Ranks|]]<br> hand size Available from start. ) and Ectoplasm (Add [[Card Modifiers#Editions|]] to a random hand size Available from start. ) both reduce hand size, which further debuffs the player along with Stuntman.

### Modifiers

- The decreased hand size doesn't allow many Gold Cards ( ), Steel Cards ( ), and Blue Seals (Creates the card for final played [[Poker Hands|]] of round if in hand<br> ) to be utilized.

### Blinds

- The Manacle ( ): -1 hand size can make assembling hands even harder.

- The Psychic ( ): Must play 5 cards. Stuntman, along with another -2 hand size debuff, can reduce hand size to 4, making the Psychic impossible to beat. If necessary, Stuntman can be sold during the round, and the Hand Size will immediately recover, with the missing cards being drawn.

## Trivia

- This Joker's appearance might be in reference to the costume worn by the American stuntman Evel Knievel.

- The "-2 hand size" effect could be a reference to how real-life stuntmen will often perform tricks hands-free, i.e., without both of their hands on the controls to their vehicles.

- Stuntman has the shortest in-game description of any Uncommon or Rare Joker, at only 23 characters.

### In other media

- Stuntman appeared as a cameo in a crossover collaboration between Balatro and Rusty's Retirement.
