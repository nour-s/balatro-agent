# The Order

Copyable Perishable Eternal

The Order is a Rare Multiplicative Mult Joker that grants X3 Mult any time you play a Straight. It can be unlocked by winning a run without playing a Straight (though playing other hands that contain straights, such as Straight Flush, are allowed).

## Synergies

### Decks

- Abandoned Deck ( Start run with no [[Card Ranks#Face cards|]] in your deck Win a run with the on [[Stakes|any difficulty]] ) and Mad World ( Extra no longer earn money<br>Earn no at end of round Eternal, Negative <br>Eternal Ranks 2 through 9 only, 32 cards totalBanned Blinds: ) both have fewer cards than usual, which makes it easier to make Straights.

- Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] )'s extra hand size makes Straights easier to find.

- 15 Minute City ( Eternal <br>Eternal Two copies of every face card;<br>no Aces, 2s, or 3s) starts with an Eternal Shortcut (Allows to be made with gaps of <br> )}} Uncommon Effect N/A (Passive) ), which allows Straights to be made with gaps in them, making it easier to get that hand.

### Poker Hands

The Order will trigger on Straights and Straight Flushes.

As all Straight hands have high Chip scaling from Planet cards, The Order can lend itself well to high-level Straight or Straight Flush runs.

### Jokers

- Four Fingers (All and can be made with 4 cards Uncommon Effect N/A (Passive) ) and Shortcut (Allows to be made with gaps of <br> )}} Uncommon Effect N/A (Passive) ) both allow Straights to be made more easily. Having either of these will make Straight builds much more reliable, making The Order easier to trigger.

- Jokers that give bonuses when Straights are played, like Crazy Joker ( ) and Devious Joker ( ), work well with The Order. Runner (Gains if played hand contains a <br> }} ) is the primary example of this, as it gives scaling Chips when the player plays the hand they already want to play if they have The Order.

- Sources of easy +Mult are useful in the early antes, such as Mystic Summit ( ), as they allow The Order to multiply a larger amount of Mult without having to level Straights first.

- Séance (If [[Poker Hands|]] is a , create a random [[Spectral_Cards|]] card <br> ), if it can be properly utilized, can be a source of Spectral cards that can help Straight builds, like Aura (Add [[Foil|]], [[Holographic|]], or [[Polychrome|]] effect to selected card in hand Available from start. ), Incantation (Destroy random card in your hand, add random [[Card_Modifiers#Enhancements|]] [[Card Ranks#Number cards|]] to your hand Available from start. )/ Familiar (Destroy random card in your hand, add random [[Card Modifiers#Enhancements|]] [[Card Ranks#Face cards|]] to your hand Available from start. ) & Grim (Destroy random card in your hand, add random [[Card Modifiers#Enhancements|]] [[Card Ranks#Aces|]] to your hand Available from start. ) (depending on which Straights one is playing more often), and Immolate (Destroys random cards in hand, gain Available from start. ) to destroy certain ranks not used in your Straights. In the same vein, Superposition (Create a [[Tarot Cards|]] card if poker hand contains an and a <br> ) gives mild synergy by giving a Tarot card from high and low Ace Straights, but this isn't very strong compared to the benefits of many other Jokers.

- Deck thinning is more useful to Straight builds than others, as adding cards doesn't necessarily make Straights more likely. For this reason, Trading Card (If of round has only card, ) and Erosion ( for each card below in your full deck <br> }} ) can form a strong combo with The Order, as the player can destroy all cards that aren't in your chosen Straight, making them more likely, and provide scaling Mult for The Order to work off of.

### Vouchers

- Wasteful () and Recyclomancy () give additional discards, allowing the player to search for Straight hands easier.

- Paint Brush ( ) and Palette ( hand size Reduce your hand size down to five cards. ) both increase hand size, which can help players with forming Straights.

- Telescope ([[Booster Packs|]] always contain the [[Planet Cards|]] card for your most played [[Poker Hands|]] ) can help upgrade Straight using Saturn (Increases hand value by and Available from start. ).

## Anti-Synergies

### Decks

- Erratic Deck ( All [[Card Ranks|]] and [[Suits|]] in deck are randomized Win a run with any [[Decks|deck]] ) can make it difficult to use The Order, as the randomized rank and suit makes it harder to form Straights.

- Five-Card Draw ( discards per round<br> hand size<br> Joker slots <br>Banned Jokers: , , )'s -3 hand size makes it extremely hard to assemble Straights.

### Jokers

- Jokers that reduce hand size, like Stuntman (,<br> hand size Earn at least 100 million (100,000,000) Chips in a single hand. ) and Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ), have anti-synergy with The Order, as a smaller hand size makes it harder to assemble Straights.

### Consumables

- Ectoplasm (Add [[Card Modifiers#Editions|]] to a random hand size Available from start. ) and Ouija (Converts all cards in hand to a single random [[Card Ranks|]]<br> hand size Available from start. ) have anti-synergy with The Order, as a smaller hand size makes it harder to assemble Straights. Not only that, but Ouija (Converts all cards in hand to a single random [[Card Ranks|]]<br> hand size Available from start. ) converts all cards held in hand into one single rank, which makes assembling Straights less consistent.

### Blinds

- The Manacle ( )'s -1 hand size makes it hard to assemble Straights.

- The Water ( ) will always start with 0 discards, making it very hard to assemble Straights.
