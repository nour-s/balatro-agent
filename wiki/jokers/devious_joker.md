# Devious Joker

Copyable Perishable Eternal

Devious Joker is a Common Chips Joker that provides +100 Chips when playing a Straight. This also includes Straight Flush and Royal Flush.

## Trivia

- This joker is part of a series that provides a Chips boost for a specific hand. The others are Sly Joker ( ), Wily Joker ( ), Clever Joker ( ), and Crafty Joker ( ).
