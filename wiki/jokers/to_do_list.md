# To Do List

is a [Poker Hand], poker hand changes at end of round Copyable Perishable Eternal
To Do List is a Common Economy joker that rewards you financially for playing specific hands. However, the hand changes after every round, so you need to be flexible.

## Synergies

### Decks

Any deck can benefit from To Do List, however the Erratic Deck ( All [[Card Ranks|]] and [[Suits|]] in deck are randomized Win a run with any [[Decks|deck]] ) might be difficult to use, due to its randomized cards that make certain hands harder to achieve.

### Poker Hands

The target hand will change randomly every round, so it doesn't synergise with any specific hand.

Much like the appearance of planet cards, it has a chance to choose secret hands if they have been played at least once during the current run.

### Jokers

- Burglar (When [[Blind|]] is selected, ) adds 3 extra hands allowing To Do list to generate more money.

- Drunkard ( discard each round Common Effect N/A (Passive) ) and Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ) add more discards which can help the player to find the hand that To Do List chose.

### Vouchers

- Grabber () and Nacho Tong (Permanently gain an additional hand per round Play a total of 2,) both give extra hands allowing To Do List to generate more money.

- Wasteful () and Recyclomancy () both give extra discards allowing the player to find the hand that To Do List chose.

## Anti-Synergies

### Jokers

- Any joker which incentivizes playing a consistent hand type throughout a run like Runner (Gains if played hand contains a <br> }} ), Supernova (Adds the number of times [[Poker Hands|]] ) or Spare Trousers (This Joker gains if played hand contains a [[Poker Hands|]] <br> }} ) has a slight anti-synergy with To Do list.

- When running Obelisk (This Joker gains per hand played without playing your most played <br> }} ), it is possible for To Do List to land on your most played hand.

- Multiple copies of To Do List () pick their hand independently (unlike e.g. Castle (This Joker gains per discarded [[Card Suits|]] card, suit changes every round <br> }} ) for which multiple copies always pick the same suit), meaning they can request different hand types.

## Trivia

- The first letters of each item on the card are an acrostic that spells "J-O-K-E-R".
