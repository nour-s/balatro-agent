# Hit the Road

discarded this round
 (Currently X1 Mult) Copyable Perishable Eternal
Hit the Road is a Rare scaling Multiplicative Mult Joker which provides an additional temporary X0.5 Mult per discarded Jack. This Joker starts with X1 Mult and is reset back to X1 Mult at the end of each round.

## Usage Notes

- If the player has multiple copies of Hit the Road, they will all level up if one Jack is discarded.

## Synergies

### Decks

- Red Deck ( ) may work well with Hit the Road thanks to the +1 discard per round it provides, allowing more Jacks to be discarded.

### Poker Hands

Hit the Road can work with any hand, but with less deck fixing, it is better to play small hands, like High Card, Pair, or Three of a Kind. This ensures that you can discard Jacks while keeping cards in hand to score.

### Jokers

- DNA (If of round has only card, ) can create 1 Jack per round (or more with copy effects), increasing the amount that Hit the Road can be scaled. It is especially beneficial to duplicate Purple Seal Jacks (Creates a card when <br> ).

- Drunkard ( discard each round Common Effect N/A (Passive) )'s +1 Discard allows more Jacks to be found and discarded.

- Juggler ( hand size Common Effect N/A (Passive) ) allows for more Jacks to be found and good cards to be held in hand for scoring.

- Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ) gives +3 Discards, allowing more Jacks to be discarded, with the caveat of -1 hand size. The hand size can make Jacks harder to find and force the player to discard fewer Jacks to hold on to scoring cards.

- Mail-In Rebate (Earn for each discarded [[Card Ranks|]], ) is one of the best Jokers to pair with Hit the Road, as the player is already most likely using all of their discards. It can generate an absurd amount of money with good deck fixing.

- Faceless Joker () is more reliable money with less deck fixing required, although it has far less potential profit than Mail-In Rebate.

- Yorick (This Joker gains every cards discarded<br>}} Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ) scales slower than Hit the Road, but its mult never resets, making it better in the long run. Once Yorick has scaled enough, Hit the Road may no longer be necessary.

- Trading Card (If of round has only card, ) can destroy cards that are not Jacks, allowing for Jacks to be found more consistently to discard. However, Trading Card takes a discard to use, so it should only be utilized if scoring is not an issue.

- Troubadour ( hand size, <br> hand each round Win consecutive rounds by <abbr title="Discards are OK to use.">playing only 1 hand</abbr> Uncommon Effect N/A (Passive) ) allows for more Jacks to be found and for scoring cards to be kept in hand. With a strong enough build, the -1 hand's downside can be minimal.

### Consumables

- Death (Select cards, convert the card into the card<br> Available from start. ) and Cryptid (Create copies of selected card in your hand Available from start. ) can be used to create more Jacks.

- The Hanged Man (Destroys up to selected cards Available from start. ) can remove non-Jack cards, allowing for Jacks to be found and made more easily.

- Medium (Add a [[Purple Seal|]] to selected card in your hand Available from start. ) and Purple Seal (Creates a card when <br> ) are best placed on Jacks with Hit the Road, as you want to discard them anyway, and the Tarot cards generated also synergize with Hit the Road.

- Strength (Increases rank of up to selected cards by Available from start. ) can be used to turn 10s (or possibly 9s) into Jacks.

### Vouchers

- Wasteful () and Recyclomancy () each give +1 discard per round, allowing for more Jacks to be discarded.

### Blinds

- The Hook ( ) discards 2 random cards every hand played, which will scale Hit the Road if the discarded card(s) are Jacks.

- If Hit the Road is debuffed on the final hand against Crimson Heart ( ), it will not reset, allowing it to scale for another round.

## Anti-Synergies

### Decks

- Abandoned Deck ( Start run with no [[Card Ranks#Face cards|]] in your deck Win a run with the on [[Stakes|any difficulty]] ), Mad World ( Extra no longer earn money<br>Earn no at end of round Eternal, Negative <br>Eternal Ranks 2 through 9 only, 32 cards totalBanned Blinds: ), and Medusa ( Eternal 52 cards, with all face cards replaced by Stone cards) don't work well with Hit the Road due to lack of initial Jacks.

- Blast Off ( hands per round<br> discards per round<br> Joker Slots Eternal <br>Eternal <br><br>Banned [[Vouchers]]: , <br>Banned Jokers: ) doesn't work well with Hit the Road, as this deck only starts with 2 discards.

### Poker Hands

Hands that work with multiple numbered cards, like Straight, Full House, Straight Flush, and Flush House, don't work well with Hit the Road because in an ideal deck you would only have Jacks; if not, you have no discards left to find these hands.

### Jokers

- Burglar (When [[Blind|]] is selected, ) removes all Discards, making Hit the Road useless.

- Any Jokers that disincentivize discarding, like Delayed Gratification (Earn per if no discards are used by end of the round Common Economy N/A (Passive) ), Banner ( ), Green Joker ( per hand played <br> per discard <br> }} ), and Ramen (, ), have poor synergy with Hit the Road.

### Blinds

- The Water ( ) starts with 0 Discards and therefore renders Hit the Road useless.

- The Mark ( ) makes it difficult to see if a given card is a Jack or another face card.

- The Plant ( ) debuffs face cards and therefore renders Hit the Road useless.

### Stakes

- Blue Stake and above hinder Hit the Road's potential because they have 1 less discard by default.

## Trivia

- The name and effect of this Joker are based on the 1961 hit song "Hit the Road Jack", written by Percy Mayfield and originally recorded by Ray Charles.

- The border for Hit the Road is a slightly different shade from all other Jokers. Hit the Road's hex code is #C2C7C9, whereas the standard is #B9C2D2.
