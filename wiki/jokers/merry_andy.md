# Merry Andy

Copyable Perishable Eternal

Merry Andy is an Uncommon Effect Joker that provides +3 discards at the cost of -1 Hand Size.

## Synergies

### Decks

Merry Andy is a good Joker early game for the economy of the Green Deck ( At end of each Round: <br> per remaining <br> per remaining <br> Earn no Discover at least items from your [[collection]] ), since it gains no interest and instead earns money only based on discards and hands remaining, extra discards are pretty valuable. Merry Andy can also be useful early game for the Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ), as its hand size penalty is completely negated by the Painted Deck's bonus hand size.

### Poker Hands

- Straights usually require more discards to set up.

- If the player needs to draw a specific enhanced card to win, Merry Andy enables the player to sift through up to 15 more cards every round.

 - Although, the reduced Hand Size makes it harder to hold multiple Steel Cards ( ) and Gold Cards ( ) at once.

### Jokers

- Juggler ( hand size Common Effect N/A (Passive) ) helps with negating this Joker's downside.

- Banner ( ) can combo with Merry Andy to provide additional chips early game.

- Castle (This Joker gains per discarded [[Card Suits|]] card, suit changes every round <br> }} ) and Yorick (This Joker gains every cards discarded<br>}} Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ) can scale much faster since Merry Andy allows you to discard 15 more cards per blind.

- It also works well with Delayed Gratification (Earn per if no discards are used by end of the round Common Economy N/A (Passive) ) for economy, increasing the payout by an extra $6. However, the smaller hand size will make it harder to play certain hands.

- Blackboard ( ) benefits from reduced hand size, making it easier to hold only Spades or Clubs for its Mult bonus.

- Raised Fist () becomes easier to use with high-ranked cards.

### Vouchers

- Paint Brush ( ) and Palette ( hand size Reduce your hand size down to five cards. ) can mitigate Merry Andy's -1 hand size penalty.

### Blinds

- If Crimson Heart ( ) disables Merry Andy, the hand size will return to normal, but the current Hand will stay intact until the next discard (e.g., 8/7). By quitting to the main menu and returning, the hand size will actually permanently increase (e.g., 8/8), giving players a permanent hand size upgrade.

### Modifiers

- Purple Seal (Creates a card when <br> ): The extra discards provides more opportunities to generate more Tarot Cards.

### Stake

- On Blue Stake and higher, Merry Andy can mitigate the -1 discard penalty.

### Jokers

- Jokers with held in hand abilities vary with Merry Andy's effectiveness.

 - Baron (), Shoot the Moon (Each held in hand gives Play every Heart card in your deck in one round. ), and Reserved Parking () are all weakened due to holding fewer face cards in hand.

 - Inversely, Raised Fist () actually benefits, as the player is less likely to hold a card with a low rank at any moment.

- Burglar (When [[Blind|]] is selected, ) always removes all discards, effectively making Merry Andy -1 Hand Size with no upside.

- Stuntman (,<br> hand size Earn at least 100 million (100,000,000) Chips in a single hand. ) adds an additional -2 hand size, making it difficult to feasibly form 5-card poker hands.

- Blue Joker ( for each remaining card in <br> }} ) generally gives fewer chips because of more discards, even thought Merry Andy reduces hand size, which Blue Joker normally benefits from.

- Green Joker ( per hand played <br> per discard <br> }} ) and Ramen (, ) discourage discarding due to weakening with each used discard.

- Mystic Summit ( ) does not work well with Merry Andy's ability.

### Consumables

- Ouija (Converts all cards in hand to a single random [[Card Ranks|]]<br> hand size Available from start. ) and Ectoplasm (Add [[Card Modifiers#Editions|]] to a random hand size Available from start. ) both reduce hand size, which further debuffs the player along with Merry Andy.

### Modifiers

- The decreased hand size doesn't allow many Gold Cards ( ), Steel Cards ( ), and Blue Seals (Creates the card for final played [[Poker Hands|]] of round if in hand<br> ) to be utilized.

### Blinds

- The Water ( ) will always set discards to 0.

## Strategy

As skipping is usually a high-risk strategy, it can be tricky to win a run in 12 or fewer rounds, meaning the player must skip at least 12 Blinds, averaging at 1.5 Blinds per Ante skipped.

- A basic Deck, such as Red Deck ( ) or Blue Deck ( hand every round Discover at least items from your [[collection]] ), is likely the best, as an additional discard/hand makes it more feasible to play more risky as is the case with skipping.

- Plasma Deck ( Balance [[Chips|]] and [[Mult|]] when calculating score for played hand <br> base Blind size Win a run with any [[Decks|deck]] ) can skip more in the early game due to its balancing ability, making the need to obtain Jokers early less of a priority.

- Anaglyph Deck ( After defeating each [[Blinds|]], gain a [[File:|24px]] [[Tags|]] Win a run with any [[Decks|deck]] ) earns the player a Double Tag ( ) upon defeating a Boss Blind, making skipping slightly more viable.

- Tags in general to look out for are Investment Tag ( ) and Speed Tag ( ), as earning Economy is critical since the player sees fewer Shops and Blind payouts overall.

- Throwback ( for each [[Blinds|]] skipped this run <br> }} ) is basically a must-have during the unlock attempt, as it directly benefits from skipping.

 - The Uncommon Tag ( ) may be a quick way to obtain Throwback early, on top of retroactively scaling it.

## Trivia

- A "merry andrew" is a term for a jester who entertains by supplying a steady stream of jokes, much like how Merry Andy supplies a steady stream of discards.1]

- If you generate Merry Andy through a Judgement (Creates a random [[Jokers|]] card<br> Available from start. ), you will get the extra discards for the current hand without cards being discarded from your hand but will drop your hand size when you next draw.

- Merry Andy has the same unlock condition as the Speedrunner achievement.
