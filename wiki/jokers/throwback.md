# Throwback

skipped this run
 (Currently X1 Mult) Copyable Perishable Eternal

Throwback is an Uncommon scaling Multiplicative Mult Joker that gains X0.25 Mult for each Blind skipped in a run. It starts at X1 Mult. This effect is retroactive, in that Blinds skipped before picking it up count towards its XMult. It is unlocked by going to the Main Menu and continuing an ongoing run.

## Synergies

### Decks

- Anaglyph Deck ( After defeating each [[Blinds|]], gain a [[File:|24px]] [[Tags|]] Win a run with any [[Decks|deck]] ) is by far the best deck to use Throwback with, as the Double Tags it generates make skipping Blinds much more appealing.

### Jokers

Jokers that have powerful up-front effects generally work great with Throwback, as they allow you to skip blinds and boost Throwback without compromising the other Jokers' scaling. Flat +Mult jokers in particular, such as Half Joker ( ), Mystic Summit ( ), Jolly Joker ( ), etc., can provide a larger base mult for Throwback, though depending on the hand type, per-card mult boosts may be stronger.

Since Throwback can boost mult to very high values, it can be worth investing in a +Chips joker, like Stuntman (,<br> hand size Earn at least 100 million (100,000,000) Chips in a single hand. ) or Bull ( for each you have <br> }} ), to improve that side of the equation.

Since Throwback prioritizes skipping blinds, consumable Jokers ( Popcorn ( <br> ), Ice Cream (<br> ), Seltzer (), and similar) go through fewer hands, allowing them to be useful for more Antes.

 Diet Cola (Sell this card to create a free [[Tags|]] Uncommon Effect N/A (Passive) ) empowers the skips you do, allowing you to double the bonus from a strong tag.

### Card Modifiers

 Glass Cards (<br> ) work very well with Throwback, as you can gain X2 Mult, and the risk to losing the cards is reduced because you are playing fewer hands.

### Vouchers

- Hieroglyph ( Ante, <br> ) and Petroglyph ( Ante, <br> ) lower the Ante by 1, giving another Small Blind and Big Blind each to skip, and thus, two more opportunities to increase Throwback's multiplier.

### Tags

- Speed Tag ( ) gives $5 for every skipped blind, which is what Throwback already incentivizes.

## Anti-Synergies

### Jokers

Because Throwback needs to skip Blinds to scale, it is very bad with Jokers that require action taken in Blinds or the shop to scale. For example, Hologram (This Joker gains every time a is added to your deck <br> }} ), Vampire (This Joker gains per scoring played, removes card <br> }} ), Supernova (Adds the number of times [[Poker Hands|]] ), Loyalty Card ( every hands played<br> ) and especially Madness (When or is selected, gain and a random Joker <br> }} ). It can be decent if a couple of blinds were skipped in the past as a temporary midgame choice, but there are often better alternatives for builds that use these kinds of scaling.

## Trivia

- The name Throwback is possibly a reference to the 80s styling of this Joker, with the arcade carpet detailing in the background lending a retro feel. It may also be a reference to the Vaporwave internet phenomenon.

- It is one of the five Jokers to have a physical card from Balatro: Special Edition. The four other jokers where that is also the case are Joker ( ), Ice Cream (<br> ), Hack (Retrigger each played , , , ) and Abstract Joker ( for each [[Jokers|]] card<br>}} ).

- It is the only Joker that is unlocked by actions performed outside of a run.
