# Rough Gem

suit earn $1 when scored Copyable Perishable Eternal

Rough Gem is an Uncommon Diamonds-oriented Economy Joker which gives the player $1 for each  Diamond scored.

## Usage Notes

Unlike unlocking Arrowhead () and Bloodstone ( ), fulfilling the condition of 30 Diamond cards in a Deck is trickier, as it doesn't allow for an easy strategy such as using the Checkered Deck ( ) for the aforementioned Jokers.

- Magic Deck ( Start run with the voucher and copies of Win a run with the on [[Stakes|any difficulty]] ) starts the player with two copies of The Fool (Creates the last [[Tarot Cards|]] or [[Planet Cards|]] card used during this run<br><small> excluded</small> Available from start. ), allowing for easier Deck manipulation early on by finding The Star (Converts up to selected cards to Available from start. ) or Death (Select cards, convert the card into the card<br> Available from start. ).

- Erratic Deck ( All [[Card Ranks|]] and [[Suits|]] in deck are randomized Win a run with any [[Decks|deck]] ) may start the player with more Diamonds than usual, making the unlock condition slightly easier.

- Any Jokers that synergize with Flush builds are likely to be useful during the unlock attempt, such as Droll Joker ( ), Crafty Joker ( ), and The Tribe ( if played hand contains a Win a run without playing a . ).

- Economy Jokers are likely to be beneficial to find more Tarots for Deck manipulation, such as Mail-In Rebate (Earn for each discarded [[Card Ranks|]], ) and Golden Joker (Earn at end of round Common Economy N/A (Passive) ).

- While Smeared Joker ( and count as the same suit, and count as the same suit Have at least [[Wild Cards|]] in your deck Uncommon Effect N/A (Passive) ) allows Heart cards to count as Diamonds for the purpose of scoring, those cards do not count as Diamonds for the purpose of unlocking the joker.

## Synergies

### Decks

- Erratic Deck ( All [[Card Ranks|]] and [[Suits|]] in deck are randomized Win a run with any [[Decks|deck]] ) can benefit from Rough Gem if the deck is randomized to have more Diamonds than other suits.

### Poker Hands

Any Flush-type hand with Diamonds benefits from Rough Gem, those being:

- Flush

- Straight Flush

- Flush House

- Flush Five

### Jokers

- Greedy Joker () provides a small boost in Mult for scoring Diamond cards, which may be useful as Rough Gem does not directly increase scoring power.

- Bull ( for each you have <br> }} ) and Bootstraps ( for every you have <br> }} Have at least Jokers at the same time. ) directly benefit as they gain scoring power based on the player's current Money.

- Jokers that require a steady Economy to scale efficiently such as Flash Card (This Joker gains per in [[The Shop|the shop]] <br> }} ) and Red Card (This Joker gains when any [[Booster Packs|]] is skipped <br> }} ) also benefit substantially.

- As Rough Gem produces money mid-round, it works very well with To the Moon (Earn an extra of for every you have at end of round Uncommon Economy N/A (Passive) ), as the money produced counts towards interest earned at the end of the round.

- Any Retrigger Joker will synergize with Rough Gem as it allows the player to earn more Money.

 - Dusk (), Hack (Retrigger each played , , , ), Seltzer (), Sock and Buskin (), or Hanging Chad ().

- Smeared Joker ( and count as the same suit, and count as the same suit Have at least [[Wild Cards|]] in your deck Uncommon Effect N/A (Passive) ) allows all  Heart cards to count as Diamonds for scoring, letting Rough Gem work on more cards.

 - Lusty Joker () and Bloodstone ( ) may be useful in this context, as Rough Gem does not provide direct scoring power.

- Splash (Every counts in scoring Common Effect N/A (Passive) ) can occasionally be useful, as the scoring of an additional Diamond that wouldn't otherwise score is extra Money.

 - However, the player ideally would be playing Flushes, which Splash is useless for.

### Consumables

- The Star (Converts up to selected cards to Available from start. ) can turn some non-Diamond cards into Diamond cards allowing the player to see Diamond cards easier and generate more money.

- The Lovers (Enhances selected card into a [[Card Modifiers|]] Available from start. ) can turn some non-Diamond cards into Wild Cards which trigger Rough Gem.

- Death (Select cards, convert the card into the card<br> Available from start. ) can turn some non-Diamond cards into Diamond cards

- The Hanged Man (Destroys up to selected cards Available from start. ) can remove some non-Diamond cards.

- Deja Vu (Add a [[Red Seal|]] to selected card in your hand Available from start. ) can apply Red Seals onto Diamond cards allowing Rough Gem to generate more money.

## Anti-Synergies

### Decks

- Checkered Deck ( ) cannot benefit from Rough Gem without adding Diamonds to the deck.

### Jokers

- While Four Fingers (All and can be made with 4 cards Uncommon Effect N/A (Passive) ) can make Flushes easier to play, it enables the player to do so with less overall cards, therefore Rough Gem will generate less money if the player opts for a 4-card Diamond Flush.

 - It may be suitable to hunt for Planet cards and/or scaling Mult Jokers if Four Fingers must remain in conjunction with Rough Gem.

### Blinds

- The Window ( ) will debuff all Diamond cards, meaning Rough Gem cannot give any Money.

- The Pillar ( ) can be detrimental as previously played Diamond cards will not be able to score, earning no Money.

## Trivia

- Rough Gem is one of four stone/mineral themed Jokers unlocked by having 30 cards of that suit in your deck. The others are Arrowhead () (Spades), Bloodstone ( ) (Hearts), and Onyx Agate () (Clubs).

- The effect of the Joker giving money could connect it to Greedy Joker, the other Joker pertaining only to Diamonds.
