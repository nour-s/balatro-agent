# Constellation

card is used
 (Currently X1 Mult) Copyable Perishable Eternal
Constellation is an Uncommon scaling Multiplicative Mult Joker that gains X0.1 Mult each time a Planet Card is used. It starts at X1 Mult. Despite leveling up the player's Poker Hands, note that the Black Hole (Upgrade every [[Poker Hands|]] by level Available from start. ) Spectral Card and the Orbital Tag ( ) do not trigger Constellation since they are not a Planet Card. This also applies to Burnt Joker (Upgrade the level of the first poker hand each round Sell 50 cards. ) and Space Joker ( chance to upgrade level of played [[Poker Hands|]] ).

## Anti-Synergies

### Jokers

- Red Card (This Joker gains when any [[Booster Packs|]] is skipped <br> }} ) wants to skip Celestial Packs instead of using them, which removes scaling for Constellation.

- Campfire (This Joker gains for each card , resets when [[Blinds|]] is defeated <br> }} ) requires selling Planet Cards instead of using them, which is the opposite of what Constellation does.

- Skipping for Throwback ( for each [[Blinds|]] skipped this run <br> }} ) or Diet Cola (Sell this card to create a free [[Tags|]] Uncommon Effect N/A (Passive) ) mean fewer Planet Cards to use, although this isn't too bad if the Skip Tag itself is the Meteor Tag ( ).
