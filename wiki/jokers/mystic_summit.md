# Mystic Summit

Copyable Perishable Eternal

Mystic Summit is a Common additive Mult Joker that gives you +15 Mult when you have 0 discards, either by using them all or through other means (such as Jokers like Burglar (When [[Blind|]] is selected, )).

In general, Mystic Summit can be useful in the early game during Ante 1 or 2, but its inability to scale makes it less useful in later Antes, and it will likely be replaced as other better Jokers are found.

## Synergies

### Decks

- Blue Deck ( hand every round Discover at least items from your [[collection]] ) lessens the consequences of using up all your discards by giving an extra hand.

### Jokers

- The strongest Joker for Mystic Summit is Burglar (When [[Blind|]] is selected, ), which trades all your discards for +3 hands. This allows Mystic Summit to be in effect as soon as you start playing.

- If the player is able to find Yorick (This Joker gains every cards discarded<br>}} Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ), Mystic Summit is a great choice as both incentivize using all of the player's discards. In addition, Mystic Summit provides + Mult to be boosted by Yorick's XMult. Yorick also mitigates Mystic Summit's anti-synergies with anything that gives extra discards.

### Vouchers

- Petroglyph ( Ante, <br> ) causes the player to lose -1 discard, making Mystic Summit easier to activate.

### Blinds

- The Water ( ): This blind synergizes with Mystic Summit as the 0 discard condition means Mystic Summit will be automatically activated.

### Stakes

- Blue Stake and above removes a discard, making Mystic Summit easier to activate.

## Anti-Synergies

### Decks

- Mystic Summit has anti-synergy with the Green Deck ( At end of each Round: <br> per remaining <br> per remaining <br> Earn no Discover at least items from your [[collection]] ), since one of the ways to get money with the Green Deck is to retain discards, and Mystic Summit is only effective when all discards are gone.

- Since Golden Needle ( Discards each cost <br><br> hands per round<br> discards per round<br>Start with Banned Jokers: <br>Banned [[Vouchers]]: , ) takes $1 from the player every time they discard, and this deck gives 3 extra discards, Mystic Summit not only requires to get rid of a lot of cards, but also costs $6 to activate here.

- Red Deck ( )'s extra discard makes it harder to activate Mystic Summit.

### Jokers

- Green Joker ( per hand played <br> per discard <br> }} ) is likely a bad choice with Mystic Summit, due to having to spend discards to have both active. This can be mitigated with Burglar (When [[Blind|]] is selected, ) but as this combo requires three Jokers it may be best to ignore it.

 - Similarly, Ramen (, ) has anti-synergy with Mystic Summit unless the player has Burglar (When [[Blind|]] is selected, ).

- Any Joker that benefits from retaining discards, such as Banner ( ) or Delayed Gratification (Earn per if no discards are used by end of the round Common Economy N/A (Passive) ) are not a good combination with Mystic Summit.

- Drunkard ( discard each round Common Effect N/A (Passive) ) and Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ) give extra discards, which makes Mystic Summit harder to activate.

### Vouchers

- Wasteful () and Recyclomancy () both give extra discards, making Mystic Summit harder to activate.
