# The Duo

Copyable Perishable Eternal

The Duo is a Rare Multiplicative Mult Joker that grants X2 Mult any time a hand containing a Pair is played. It can be unlocked by winning a run without playing a Pair, though playing other hands that contain pairs, such as Three of a Kind, are allowed.

Since The Duo offers only X2 Mult and has no scaling potential, it has limited utility past Ante 8.

## Usage Notes

Winning a run without playing a Pair is not difficult as hands played that contain Pairs do not count towards the unlock requirement. Nonetheless, there is a clear winner in which Deck to use to unlock The Duo:

- Checkered Deck ( ) allows the player to play Flushes consistently throughout the run, being able to avoid Pairs a large majority of the time.

The played Pair does not need to score for The Duo to activate. The only way to have this happen is when playing a Flush with Four Fingers (All and can be made with 4 cards Uncommon Effect N/A (Passive) ) that also has a differently suited Pair (ex: 10 9 4 3 of Spades and 10 of Diamonds).

The Duo can also activate on a Straight with Four Fingers when there is a repeated rank (ex: 10 10 9 8 7).

Revealing a pair of matching ranks via Vampire (This Joker gains per scoring played, removes card <br> }} ) removing Stone (<br>) from a card will not activate The Duo; the determination is made when the hand is played, not when the Joker effect activates.

## Synergies

### Decks

All Decks work equally well, though Erratic Deck ( All [[Card Ranks|]] and [[Suits|]] in deck are randomized Win a run with any [[Decks|deck]] ) is slightly better, because the odds of a finding a Pair are higher given the unequal distribution of the Deck.

### Poker Hands

Although only a X2 Mult boost, this Joker can trigger on every hand except High Card:

- Pair, Two Pair

- Three of a Kind, Four of a Kind, Five of a Kind

- Full House, Flush House

- Flush Five

- Flush, when a pair is present in the played hand

- Straight and Straight Flush, when containing a pair using Four Fingers (All and can be made with 4 cards Uncommon Effect N/A (Passive) ) (ex: 10 9 9 8 7)

Any Joker that provides benefits to hands that contain Pairs provide direct synergy, such as:

## Trivia

The Duo has a slightly different background to the other Rare Hand Jokers.

- This Joker is part of a set that gives strong Mult boosts for playing specific hands, which are unlocked by winning a run without playing that hand. The others are The Trio ( if played hand contains a Win a run without playing a . ), The Family ( if played hand contains a Win a run without playing a . ), The Order ( if played hand contains a Win a run without playing a . ) and The Tribe ( if played hand contains a Win a run without playing a . ).

 - Though all 5 Jokers have identical, color-swapped backgrounds, The Duo has small portions in the center that differ from on every other in the set.

- This Joker has similar colors to Jolly Joker ( ) and Sly Joker ( ), two other Jokers that give bonuses if the played hand contains a Pair.
