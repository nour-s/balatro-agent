# Castle

card, suit changes every round
 (Currently +0 Chips) Copyable Perishable Eternal

Castle is an Uncommon scaling Chips Joker that adds Chips to played hands. The amount of Chips added begins at +0 and is permanently increased by +3 for each time a card of the specified suit is discarded.

The suit is chosen proportionally to the suit composition of the current deck and changes each round. For example, in a deck of 20  Clubs and 10  Spades, Castle would select  Clubs two-thirds of the time, and  Spades for the remaining third. If the deck has no card with any suit (i.e., full of Stone Cards (<br>)) Castle will default to +3 Chips per discarded  Spade.

## Synergies

### Decks

- The Checkered Deck ( ) works well with Castle due to the lower suit diversity.

- The Red Deck ( ) can use its +1 discard for more scaling.

- The Plasma Deck ( Balance [[Chips|]] and [[Mult|]] when calculating score for played hand <br> base Blind size Win a run with any [[Decks|deck]] ) gets more value from the chips than any other deck.

- The Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ) increases the player's hand size by two, which allows players to see and discard more cards desired by Castle.

### Poker Hands

- Because Castle scales based on discarding cards, it means that weaker, easier hands like High Card and Pair allow for the effect to be used more consistently.

- Alternately, when players manage to shape their deck down to a single suit, Flush-based hands like Flush, Straight Flush, Flush House, and Flush Five become much more viable.

### Jokers

- Drunkard ( discard each round Common Effect N/A (Passive) ) and Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ) both provide additional discards, allowing Castle to scale more per round.

- Yorick (This Joker gains every cards discarded<br>}} Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ) will also scale alongside Castle.

- Smeared Joker ( and count as the same suit, and count as the same suit Have at least [[Wild Cards|]] in your deck Uncommon Effect N/A (Passive) ) makes Castle twice as easy to scale.

### Vouchers

- Paint Brush ( ) and Palette ( hand size Reduce your hand size down to five cards. ) increase the player's hand size, allowing players to see more of their deck and give them more chances to discard cards chosen by Castle.

- Wasteful () and Recyclomancy () both give 1 extra discard, allowing Castle to scale faster.

### Blinds

- The Hook ( ) automatically discards 2 cards held in hand, which can trigger Castle's scaling if one or both cards are of the chosen suit.

### Tarot Cards

- Wild Cards () count towards any suit as the bonus, so The Lovers (Enhances selected card into a [[Card Modifiers|]] Available from start. ) can be helpful.

- Removing all cards of a certain suit using tarot cards gives Castle more chance to trigger (like Checkered Deck ( )).

- Green Deck ( At end of each Round: <br> per remaining <br> per remaining <br> Earn no Discover at least items from your [[collection]] ) incentivizes not discarding, as it basically means throwing money away, which can make Castle difficult to use.

### Jokers

- Burglar (When [[Blind|]] is selected, ) prevents this Joker entirely from scaling.

- Banner ( ), Delayed Gratification (Earn per if no discards are used by end of the round Common Economy N/A (Passive) ), Green Joker ( per hand played <br> per discard <br> }} ), and Ramen (, ) conflict with this Joker, as their effects rely on not using discards.

- Similarly, Trading Card (If of round has only card, ) and Burnt Joker (Upgrade the level of the first poker hand each round Sell 50 cards. ) place restrictions on discards, which interferes with triggering Castle's effect optimally.

### Blinds

- The Goad ( ), The Head ( ), The Club ( ), and The Window ( ) all debuff a specific suit, preventing Castle from scaling if it happens to pick the to-be-debuffed suit upon entering the Boss Blind. One trick to avoid this is to check Castle's chosen suit before entering Big Blind: if it's not the to-be-debuffed suit, skip the Blind to ensure it can scale normally during the boss blind.

- The Water ( ) will always set your discards to 0, meaning Castle Joker does not scale.

- The House ( ), The Fish ( ), The Wheel ( ) and The Mark ( ) all draw cards face down, which makes it difficult to tell if the cards are the same suit as the one that Castle wants to discard.

### Stakes

- Blue Stake and above hinder Castle's potential because they have 1 less discard by default.
