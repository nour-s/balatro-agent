# Smeared Joker

and  Diamonds count as the same suit,  Spades and  Clubs count as the same suit Copyable Perishable Eternal

Smeared Joker is an Uncommon Effect Joker which makes  Diamonds and  Hearts count as the same suit, and  Clubs and  Spades count as the same suit. This effect applies to all Jokers and Blinds which have an effect based on suit.

To unlock this Joker, the player must have at least 3 Wild Cards () in the deck.

## Synergies

### Decks

- Black Deck ( Joker slot <br> hand every round Discover at least items from your [[collection]] ): The +1 Joker Slot allows Smeared Joker to function as a utility Joker while still providing space for 5 other Jokers to score.

- Checkered Deck ( ) is split between two suits that do not combine due to the Smeared Joker, so it loses out on some benefits. However, it allows the deck to benefit from effects that involve  Clubs and  Diamonds, including very powerful Jokers like Ancient Joker (Each played card with [[Card Suits|]] gives when scored, <br> <small>suit changes at end of round</small> ), so it can still be quite valuable.

- Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] )'s hand size of 10 guarantees a Flush with Smeared Joker and no Stone Cards (<br>). However, with Painted Deck's fewer Joker slots, keeping effect Jokers like Smeared Joker can be risky.

### Poker Hands

Works best with the following hands if you are trying to go for a flush build.

- Straight Flush

- Flush House

- Wrathful Joker () and Gluttonous Joker () - Each  Club or  Spade provides +3 Mult.

There are other Jokers that work together well with Smeared Joker:

- Jokers that are effective with Flushes, such as Droll Joker ( ), Crafty Joker ( ), and The Tribe ( if played hand contains a Win a run without playing a . ) all become easier to activate.

### Vouchers

- Paint Brush ( ) once again increases hand size, guaranteeing a Flush with Smeared Joker under most circumstances.

- Palette ( hand size Reduce your hand size down to five cards. )'s increase to 10 is less valuable since Flushes are already guaranteed, but still gives more room to find specific cards if that's desirable for some reason.

## Anti-Synergies

### Blinds

Smeared Joker doubles a player's vulnerability to suit bosses:

- The Head ( ) and The Window ( ) both debuff all  Diamond and  Heart cards.

- The Club ( ) and The Goad ( ) both debuff all  Club and  Spade cards.

## Trivia

- This card's name and design are a reference to how in a normal playing card deck,  Club and  Spade are the same color, just like  Diamond and  Hearts. As such, if the actual symbol of the suit was smeared and illegible, it would be difficult to tell them apart.

- Due to the letters being smeared, this Joker is the only non-Legendary Joker to not legibly contain the word "Joker" in its sprite.
