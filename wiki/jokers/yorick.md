# Yorick

[23] cards discarded
(Currently X1 Mult) Copyable Perishable Eternal

Yorick is one of the five Legendary Jokers that can be generated from using The Soul (Creates a [[Legendary Jokers|]] Joker <br> Available from start. ) Spectral card, which in itself has a 3 in 1000 (0.3%) chance to replace any given card in a Spectral Pack or Arcana Pack.

Yorick starts at X1 Mult and gains an extra X1 Mult for every 23 cards discarded.

## Synergies

### Decks

- Red Deck ( ) starts with an extra discard, which can help Yorick by allowing up to 5 more cards to be discarded than normally available.

- Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ) starts with a larger Hand size, letting the player get rid of unused cards to their heart's content while still being able to keep the important cards to form poker hands.

- Five-Card Draw ( discards per round<br> hand size<br> Joker slots <br>Banned Jokers: , , ) works very well with Yorick, as this deck starts with 6 discards, which means it can consistently gain X1 Mult every blind with the use of 5 discards.

- Golden Needle ( Discards each cost <br><br> hands per round<br> discards per round<br>Start with Banned Jokers: <br>Banned [[Vouchers]]: , ) may also work with Yorick, as this deck also starts with 6 discards. However, this deck penalizes the player by losing $1 for each discard, which can make it awkward to use Yorick.

### Poker Hands

Yorick can be made to work with all poker hands. Discarding a large amount of cards to form bigger poker hands like Straight or Full House is a viable strategy, while smaller poker hands like High Card and Pair can still be made consistently after discarding for Yorick.

### Jokers

- Drunkard ( discard each round Common Effect N/A (Passive) ) gives 1 extra discard, which is the same as using Red Deck ( ), making scaling Yorick much easier.

- Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ) gives 3 extra discards, which drastically helps improving Yorick, almost always giving at least an extra X1 Mult to it every round if all discards are used to their maximum effect. Though, it is important to note that its downside, -1 Hand size, along with it being an Uncommon, makes it more difficult to come by than Drunkard ( discard each round Common Effect N/A (Passive) ).

- A large Hand size is typically preferred when discarding for Yorick to keep important cards, which makes Jokers that give more Hand sizes, like Juggler ( hand size Common Effect N/A (Passive) ) and Troubadour ( hand size, <br> hand each round Win consecutive rounds by <abbr title="Discards are OK to use.">playing only 1 hand</abbr> Uncommon Effect N/A (Passive) ), very helpful when paired with Yorick.

- Mystic Summit ( ), Faceless Joker (), Mail-In Rebate (Earn for each discarded [[Card Ranks|]], ), Castle (This Joker gains per discarded [[Card Suits|]] card, suit changes every round <br> }} ), Hit the Road (This Joker gains for every discarded this round<br> }} Discard at the same time. ), and Burnt Joker (Upgrade the level of the first poker hand each round Sell 50 cards. ) all have abilities that activate when using discards.

### Spectral Cards

- Medium (Add a [[Purple Seal|]] to selected card in your hand Available from start. )'s Purple Seal (Creates a card when <br> ) generates a Tarot Card after discarding a card. If the player is discarding left and right to beef up Yorick, it is helpful to get another bonus out of it.

### Vouchers

- Wasteful () and Recyclomancy () both give 1 extra discard, allowing Yorick to scale faster.

### Blinds

- The Hook ( )'s forced discard effect can help with scaling Yorick.

### Jokers

- Burglar (When [[Blind|]] is selected, ) shuts down Yorick by removing all discards and is highly recommended to be avoided at all costs when using Yorick, especially when it has the Eternal () sticker.

- Delayed Gratification (Earn per if no discards are used by end of the round Common Economy N/A (Passive) ), Banner ( ), Green Joker ( per hand played <br> per discard <br> }} ), and Ramen (, ) all incentivize not discarding, which goes against Yorick.

- Trading Card (If of round has only card, ) has less synergy with Yorick, since it incentivizes only discarding 1 card for the first discard.

### Blinds

- The Water ( ) gets rid of all discards, meaning Yorick is unable to scale during the blind.

### Stakes

- Blue Stake and above hinder Yorick's potential because they have 1 less discard by default.

## Trivia

- Yorick is an unseen character in William Shakespeare's tragedy Hamlet. He is the dead court jester whose skull is exhumed in Act 5, Scene 1 of the play. The sight of Yorick's skull evokes a reminiscence by Hamlet of the man, who apparently played a role during Hamlet's upbringing: "Alas, poor Yorick! I knew him, Horatio; a fellow of infinite jest."

 - This also seems to be the origin of the 23 cards needed to discard for Yorick's scaling, as in the same Act 5, Scene 1 of the play, the following lines can be found: "This skull has lain in the earth three-and-twenty years ... Sir, this skull belonged to Yorick, the king's jester".1]

 - Yorick's costume in Balatro seems to be based on the jester's appearance in the 1868 painting The Young Lord Hamlet by Philip Hermogenes Calderon, albeit with the primary color altered from red to yellow.

- This joker is animated, with Yorick hovering just above the gold card face.
