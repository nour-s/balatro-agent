# Gluttonous Joker

suit give +3 Mult when scored Copyable Perishable Eternal

Gluttonous Joker is a Common Joker that provides conditional Additive Mult to played hands. One of the four 'sinful' Jokers made to support specific Card Suits, it gives +3 Mult each time a  Club card is scored. If the played card is retriggered, either by a Joker effect or by its own Red Seal (), Gluttonous Joker's effect will activate again.

- Magic Deck ( Start run with the voucher and copies of Win a run with the on [[Stakes|any difficulty]] )'s Fool Tarots can be used to mass-convert cards to have Club suits.

- Ghost Deck ( [[Spectral Cards|]] cards may appear in the shop, start with a card Win a run with the on [[Stakes|any difficulty]] ) can find a Sigil (Converts all cards in hand to a single random [[Card Suits|]] Available from start. ) Spectral card in the shop, which may convert cards into Clubs.

- Certain Erratic Deck ( All [[Card Ranks|]] and [[Suits|]] in deck are randomized Win a run with any [[Decks|deck]] ) configurations can have more  Clubs.

### Poker Hands

This Joker works best with Flush hands, if they contain only Club suit cards. These hands are as follows:

- Flush

- Straight Flush (and Royal Flush)

- Flush House

- Flush Five

Other hands that score more cards are generally better to play with Gluttonous Joker, as they allow more Club cards to be scored.

### Jokers

- As Gluttonous Joker's effect triggers when a Club card is scored, retrigger Jokers that allow cards to be scored multiple times are very good to combo with Gluttonous.

- Flush Jokers, like Crafty Joker ( ) and Droll Joker ( ), work well with Gluttonous Joker, as you want to be playing flushes to maximize its effect anyways.

- Onyx Agate () also triggers off of Club cards scored, forming a natural synergy with Gluttonous Joker. As Onyx Agate is a strict upgrade to Gluttonous, it can be used as a replacement, but it also can be used in builds that have high X Mult from hands from Jokers like, sometimes, Ancient Joker (Each played card with [[Card Suits|]] gives when scored, <br> <small>suit changes at end of round</small> ), or The Idol (Each played [[Card Ranks|]] of [[Card Suits|]] gives when scored<br><small>Card changes every round</small> In one hand, ). This combo also benefits from a strong Chip Joker, ideally one that also benefits from Flushes. Castle (This Joker gains per discarded [[Card Suits|]] card, suit changes every round <br> }} ) is the best example of this kind of Joker.

- Because Gluttonous Joker's Mult is applied during the scoring of your played cards, Jokers that focus on creating Tarot cards are helpful to Gluttonous Joker's viability. They can both allow more Clubs to be created, and create certain Enhancements (particularly Glass and Steel) that work well with high hand-scoring Mult builds.

- Smeared Joker ( and count as the same suit, and count as the same suit Have at least [[Wild Cards|]] in your deck Uncommon Effect N/A (Passive) ) effectively turns all Spade cards into Clubs for the purpose of Gluttonous Joker's effect.

## Anti-Synergies

### Decks

- Checkered Deck ( ) starts with no Clubs, making Gluttonous Joker less useful.

### Blinds

- When facing The Club ( ), all Club cards are debuffed, meaning Gluttonous Joker cannot activate.

## Trivia

Old design of Gluttonous Joker

- Gluttony is one of the traditional seven deadly sins. The four basic Jokers that provide +3 Mult for each suit are each based on these. The other Sinful Jokers are Greedy Joker (), Lusty Joker () and Wrathful Joker ().

 - Gluttonous Joker's uncommon counterpart, Onyx Agate (), gives +7 Mult instead of +3 Mult, which could perhaps play into the "acquiring things especially beyond the point of necessity" definition of gluttony.

- Unlike the other Sinful Jokers, there appears to be no correlation between the suit and the sin chosen.

- In the game's code, Gluttonous Joker is misspelled as "gluttenous".

- In demo versions of the game, the color of Gluttonous Joker was dark blue instead of green. This was changed in the release version (update 1.0.0h) to match the color of the suit better.
