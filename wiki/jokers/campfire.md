# Campfire

, resets when Boss Blind is defeated
 (Currently X1 Mult) Copyable Perishable Eternal

Campfire is a Rare scaling Multiplicative Mult Joker that gains X0.25 Mult per sold card, which can theoretically be scaled infinitely. However, it resets itself to X1 Mult after defeating a Boss Blind.

Selling any Joker or consumable (Tarot, Planet, or Spectral card) will add X0.25 Mult to Campfire's bonus. They must be deliberately sold—destroying them through use of other cards (such as Hex (Add [[Card Modifiers#Editions|]] to a random [[Jokers|]], destroy all other Jokers Available from start. ) or by the effect of Ceremonial Dagger (When [[Blinds|]] is selected, destroy [[Jokers|Joker]] to the right and permanently add its sell value to this <br> }} )) does not count.

Note that unlike most non-retroactive positively scaling jokers, Campfire can be Perishable (), likely due to the fact that it will reset at least once before it perishes.

## Synergies

### Decks

- Zodiac Deck ( Start run with , , and Win a run with any [[Decks|deck]] ) starts you with the Tarot Merchant and Planet Merchant Vouchers, which make consumable cards appear more often, and the Overstock Voucher, which makes more cards appear in the shop. This combines to provide a vast amount of cards you can sell every shop.

- Yellow Deck ( Start with extra Discover at least items from your [[Collection|collection]] ) lets you build money much faster, which is needed to scale Campfire.

- Black Deck ( Joker slot <br> hand every round Discover at least items from your [[collection]] )'s extra Joker slot lets you have five Jokers while still having room to buy and sell with the final slot.

### Poker Hands

- Since Campfire provides a strong XMult bonus, any Poker Hand with a high base Mult like Flush, Straight, Full House, will benefit greatly.

### Jokers

### Consumables

- All consumables will work with campfire, however The Emperor (Creates up to random cards<br> Available from start. ) and The High Priestess (Creates up to random [[Planet Cards|]] cards<br> Available from start. ) work particularly well, since they both create 2 cards to sell.

### Vouchers

- Clearance Sale () and Liquidation () make it cheaper to constantly buy and sell cards, the latter reducing the purchase and sell value of all consumables to 1 and effectively allowing you to scale Campfire for free.

- Overstock ( ) and Overstock Plus ( ) make more cards appear in the shop to buy and sell.

- Reroll Surplus () and Reroll Glut () make it cheaper to reroll to find more cards to sell.

- Tarot Merchant ( ), Planet Merchant ( ), Tarot Tycoon ( ) and Planet Tycoon ([[Planet Cards|]] ) make Tarot and Planet cards appear in shop more frequently, allowing repeated buying & selling them.

- Crystal Ball ( ) provides +1 Consumable Slot, which allows the player to hold 2 Consumables while still having a room to sell.

- Antimatter ( ) provides +1 Joker Slot, which allows the player to hold 5 Jokers while still having a room to sell.

### Blinds

- If Crimson Heart ( ) disables Campfire on the last hand the player needs to win, Campfire will not reset, allowing it to be scaled for another Ante.

### Modifiers

- Purple Seal (Creates a card when <br> ) can generate some Tarot Cards which can be sold if they are not necessary.

- Blue Seal (Creates the card for final played [[Poker Hands|]] of round if in hand<br> ) can generate some Planet Cards at the end of the round, allowing Campfire to scale faster.

### Stakes

- On Gold Stake buying a Rental () Joker and selling it immediately thereafter allows you to scale Campfire with no economic downside.

### Decks

- Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] )'s -1 Joker Slot and Nebula Deck ( Start run with the voucher <br> consumable slot Win a run with the on [[Stakes|any difficulty]] )'s -1 Consumable Slot can make it harder to have a room for Jokers to sell and hold onto Consumables to sell, respectively, to increase its effectiveness between antes.

- Ghost Deck ( [[Spectral Cards|]] cards may appear in the shop, start with a card Win a run with the on [[Stakes|any difficulty]] ) allows for Spectral Cards to appear in the shop, but they're more expensive than Tarots or Planets, and selling them is not always something the player would want to do.

### Jokers

- Fortune Teller ( per [[Tarot Cards|]] card used this run <br> }} ) and Constellation (This Joker gains every time a [[Planet_Cards|]] card is used <br> }} ) encourage the player to use Tarot Cards and Planet Cards, respectively, which is the opposite of what Campfire does.

- Throwback ( for each [[Blinds|]] skipped this run <br> }} ) and Diet Cola (Sell this card to create a free [[Tags|]] Uncommon Effect N/A (Passive) ) rely on skipping blinds, which results in Campfire being reset faster.

## Trivia

- Like a real campfire, this Joker needs a continuous source of fuel to keep burning brightly. This Joker's art also references how the selling animation looks like the sold card is burning.

- Campfire is the only Joker with a buy price of $9.
