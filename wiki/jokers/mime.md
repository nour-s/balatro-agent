# Mime

Copyable Perishable Eternal

Mime is an Uncommon Retrigger Joker that can retrigger all held in hand effects (cards that are not played but remain in hand).

Mime can retrigger Gold ( ) and Steel Cards ( ), as well as Blue Seals (Creates the card for final played [[Poker Hands|]] of round if in hand<br> ). Mime will also retrigger other Jokers that activate from held in hand cards, such as Baron (). It is important to note that Mime will not reactivate a Red Seal () a second time. For example, a Red Seal Gold Card, will only activate three times with Mime, giving a total of $9 after the round ends.

## Synergies

### Decks

- The Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ) gives an additional +2 to the player's hand size, allowing Mime to activate more cards.

- The Ghost Deck ( [[Spectral Cards|]] cards may appear in the shop, start with a card Win a run with the on [[Stakes|any difficulty]] ) allows for Spectral Cards to be readily available in the Shop, making Seals more attainable.

### Poker Hands

- High Card is arguably the best hand with Mime, since it requires the least amount of cards to play. This allows for more cards to be held in hand.

- Pair and Three of a Kind can be used for the same reasons, although slightly less effectively.

### Jokers

- Held in hand Jokers synergize the best with Mime, since it enables them to be activated twice. These include:

 - Reserved Parking () will generally give $0.25 more on average.

 - This can change depending on retriggers and held in hand effects, see the Reserved Parking page for an explanation on the specific interactions and numbers.

 - Raised Fist () and Shoot the Moon (Each held in hand gives Play every Heart card in your deck in one round. ): Provides double their + Mult to the score.

 - Baron (): Potentially the strongest duo with Mime by giving an additional X1.5 Mult for each retrigger. This, paired with the Plasma Deck ( Balance [[Chips|]] and [[Mult|]] when calculating score for played hand <br> base Blind size Win a run with any [[Decks|deck]] ), can also allow the player to possibly reach the elusive Naneinf score.

- Steel Joker (Gives for each [[Steel Cards|]] in your <br> }} ): Steel Cards ( ) are exceptionally strong with Mime, and attaining more Steel cards will also scale this Joker.

- Constellation (This Joker gains every time a [[Planet_Cards|]] card is used <br> }} ): Blue Seals (Creates the card for final played [[Poker Hands|]] of round if in hand<br> ) are retriggered by Mime, creating more planets and scaling this Joker faster.

- Tarot generating Jokers (ex. 8 Ball ( chance for each played to create a [[Tarot Cards|]] card when scored<br> ), Cartomancer (Create a [[Tarot Cards|]] card when [[Blinds|]] is selected <br> Discover every Tarot Card. ), etc.) can help maximize the effectiveness of Mime by providing the necessary means to manipulate the deck and modify cards.

 - Vagabond (Create a [[Tarot Cards|]] ) can benefit the most from Mime, since it can activate Gold Cards ( ) a second time after the round ends, allowing the for player to still maintain their economy without hampering Vagabond's trigger condition.

- Having a larger hand size allows for more cards to be held in hand. Jokers like Turtle Bean ( hand size, reduces by each round Uncommon Effect N/A (Passive) ), Juggler ( hand size Common Effect N/A (Passive) ) and Troubadour ( hand size, <br> hand each round Win consecutive rounds by <abbr title="Discards are OK to use.">playing only 1 hand</abbr> Uncommon Effect N/A (Passive) ) can be beneficial to have.

- Blueprint (Copies ability of [[Jokers|]] )/ Brainstorm (Copies the ability of leftmost [[Jokers|]] Discard a . ): If Blueprint or Brainstorm is copying a different joker when a hand is played, and then moved to copy Mime as the hand is being scored, the Blueprint/Brainstorm will copy Mime when it retriggers Blue Seals/Gold Cards.

- Hanging Chad () can help retrigger modified played cards, making them more effective, especially so since smaller hands are generally used with Mime.

### Vouchers

- Paint Brush ( ) and Palette ( hand size Reduce your hand size down to five cards. ) each increase your hand size by +1, allowing for more cards to be held in hand.

### Playing Cards

Mime works extraordinarily well with Gold Cards ( ), Steel Cards ( ) and cards with Blue Seal (Creates the card for final played [[Poker Hands|]] of round if in hand<br> ) as it is able to retrigger their effects.

## Anti-Synergies

### Decks

- Since 3 of the 4 Jokers that utilize held in hand abilities trigger with face cards, the Abandoned Deck ( Start run with no [[Card Ranks#Face cards|]] in your deck Win a run with the on [[Stakes|any difficulty]] ) and Medusa ( Eternal 52 cards, with all face cards replaced by Stone cards) can be difficult to use with Mime.

- Five-Card Draw ( discards per round<br> hand size<br> Joker slots <br>Banned Jokers: , , ) does not work well with this Joker due to the decreased hand size and banning of hand size-increasing Jokers.

### Jokers

- Mime suffers from Jokers that reduce hand size, such as Stuntman (,<br> hand size Earn at least 100 million (100,000,000) Chips in a single hand. ) and Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ), due to them lowering the number of cards with "Held in Hand" abilities able to be active at once.

- Notably, Mime does not affect Blackboard ( ) despite its ability having the "held in hand" description.

- Golden Ticket (Played [[Gold Cards|]] ) encourages the player to play Gold Cards instead of keeping them in hand, making Mime weaker.

### Blinds

- The Hook ( ) activates before Mime does, and can remove desirable cards from a player's hand during scoring.

## Trivia

- The word "Mime" comes from the Greek word "mimos" (μῖμος) which means "imitator" or "actor", as well as the Greek adjective "mimikos" (μιμικός) which roughly translates to "of or pertaining to imitation/mimicry". This is well reflected in the Mime's ability to imitate the effects of some card modifiers.

- Although mimes originated in Ancient Greece and Rome, Mime's design is based on the modern French mime, likely because that depiction is more universally recognized. This is reflected in the card's artwork, which features stereotypical French elements such as a beret and a red scarf.
