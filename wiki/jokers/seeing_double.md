# Seeing Double

Club card and a scoring card of any other suit Copyable Perishable Eternal

Seeing Double is an Uncommon Multiplicative Mult Joker that gives X2 Mult when a hand is scored with one  Club card and any other suit card. Seeing Double will not activate if all  Club cards or all non  Club cards are debuffed.

## Usage Notes

Unlocking Seeing Double may prove to be challenging at first due to its condition being oddly specific towards a specific rank and suit. In particular, there are no Jokers that directly have synergies with 7s, the sole exception being Odd Todd (Played cards with [[Card Ranks|]] rank give when scored <br> ). Nevertheless, there are two Decks that are ideal to use when attempting the unlock:

- Magic Deck ( Start run with the voucher and copies of Win a run with the on [[Stakes|any difficulty]] ) starts with two copies of The Fool (Creates the last [[Tarot Cards|]] or [[Planet Cards|]] card used during this run<br><small> excluded</small> Available from start. ), allowing the player to copy Tarots to make more 7 of Clubs, such as Death (Select cards, convert the card into the card<br> Available from start. ), The Moon (Converts up to selected cards to Available from start. ), or Strength (Increases rank of up to selected cards by Available from start. ).

- Erratic Deck ( All [[Card Ranks|]] and [[Suits|]] in deck are randomized Win a run with any [[Decks|deck]] ) may start the player with a Deck that contains multiple copies of a 7 of Clubs, making it easier to create more copies or, in some cases, require no further modifications.

## Synergies

### Decks

- Erratic Deck ( All [[Card Ranks|]] and [[Suits|]] in deck are randomized Win a run with any [[Decks|deck]] ) may start with more Clubs than a standard deck, which could activate Seeing Double's condition more easily.

### Poker Hands

- Straight and Full House both generate X2 Mult a large majority of the time, making Seeing Double great for those strategies.

### Jokers

- Smeared Joker ( and count as the same suit, and count as the same suit Have at least [[Wild Cards|]] in your deck Uncommon Effect N/A (Passive) ) means  Clubs can also count as  Spades and vice-versa. This means that playing any hand (even High Card) containing a Club or Spade will trigger Seeing Double.

 - This interaction might be a bug. This interaction also no longer applies when the card is Wild ().

- Splash (Every counts in scoring Common Effect N/A (Passive) ) allows triggering the bonus by adding an extra card that would otherwise be ineligible for scoring; e.g. ♣️A ♣️A ♣️A ♣️A ♥️4.

- Almost any hand that triggers Flower Pot ( if poker hand contains a card, card, card, ) will also trigger Seeing Double, providing a total of X6 Mult.

 - If some cards are Debuffed, Flower Pot ( if poker hand contains a card, card, card, ) may trigger without Seeing Double, as Seeing Double requires at least 1 non debuffed  Club and at least 1 non debuffed non  Club, while Flower Pot ( if poker hand contains a card, card, card, ) still counts debuffed cards.

### Card Modifiers

- Wild Cards () can count as either the Club or the non-Club, but not both. Generally this means adding any second scoring card (including a second Wild Card) will trigger the bonus, though Stone Cards (<br>) will not qualify, having no suit at all.

## Anti-Synergies

### Decks

- Checkered Deck ( ) makes it significantly harder to activate Seeing Double, as it contains no Club cards naturally.

### Poker Hands

- Flush is hard to use, requiring a Wild Card () or an off-suit card via Four Fingers (All and can be made with 4 cards Uncommon Effect N/A (Passive) ).

### Blinds

- The Club ( ) renders Seeing Double useless, as all Club Cards are debuffed and debuffed Club cards will not trigger Seeing Double.

- The Pillar ( ) can make Seeing Double difficult to trigger if players already played some Club cards in the previous round, making them debuffed upon facing this boss blind. This can be remedied by carrying Tarot cards that provide enhancements into the boss blind to "revive" debuffed Club Cards.

## Trivia

- LocalThunk has said the name of this Joker is based on the album Seeing Double from 2002 by the British pop group S Club 7.1]

 - Its effect in demo version used to be, and its current unlock condition is still related to 7 of Clubs.
