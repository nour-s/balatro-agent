# Scary Face

Copyable Perishable Eternal

Scary Face is a Common Chips Joker that adds +30 Chips for each scored Face Card in a hand.

## Synergies

### Decks

- Erratic Deck ( All [[Card Ranks|]] and [[Suits|]] in deck are randomized Win a run with any [[Decks|deck]] ) can synergize with Scary Face if it's randomized to have lots of Face Cards.

- Red Deck ( )’s extra discard can be used to dig for more face cards in the player’s deck.

- Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ) has a larger hand size, allowing for more face cards to appear. However, the -1 Joker slot means Scary Face may not be as viable as some other options.

- Plasma Deck ( Balance [[Chips|]] and [[Mult|]] when calculating score for played hand <br> base Blind size Win a run with any [[Decks|deck]] ) can use the large influx of Chips to win a large number of early hands, allowing one to search for better Jokers easier.

- Mad World ( Extra no longer earn money<br>Earn no at end of round Eternal, Negative <br>Eternal Ranks 2 through 9 only, 32 cards totalBanned Blinds: ) starts with an Eternal Pareidolia (All cards are considered cards Uncommon Effect N/A (Passive) ), which causes Scary Face to trigger on all played cards.

### Poker Hands

Scary Face requires face cards (King, Queen, and Jack) to trigger, so hands with more face cards in them will allow Scary Face to award more Chips. For example, a Full House consisting of K K K Q Q will trigger Scary Face five times.

### Jokers

- If the first card of the hand is a face card, Hanging Chad () will cause Scary Face to trigger thrice, giving +90 Chips.

- Pareidolia (All cards are considered cards Uncommon Effect N/A (Passive) ) causes Scary Face to trigger on all played cards, effectively making every card a Bonus Card ( ).

- Sock and Buskin () will add an additional +30 Chips per face card scored.

- Triboulet (Played Kings and Queens each give when scored Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ) provides a Mult multiplier for each played King or Queen.

- Business Card () gives each face card played a 50% chance to provide $2 when scored.

- Smiley Face () provides additional Mult for each face card played.

- Splash (Every counts in scoring Common Effect N/A (Passive) ) allows non-scoring cards to trigger Scary Face (by making them score).

## Anti-Synergies

### Decks

- Abandoned Deck ( Start run with no [[Card Ranks#Face cards|]] in your deck Win a run with the on [[Stakes|any difficulty]] ) and Medusa ( Eternal 52 cards, with all face cards replaced by Stone cards) are not recommended due to lack of Face cards.

### Jokers

- Baron (), Shoot the Moon (Each held in hand gives Play every Heart card in your deck in one round. ), Hit the Road (This Joker gains for every discarded this round<br> }} Discard at the same time. ) and Reserved Parking () encourage not playing face cards.

- Ride the Bus (This Joker gains per hand played without a scoring card <br> }} ) actively punishes playing face cards.

- Canio (This Joker gains when a card is destroyed <br> }} Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. )'s ability activates only upon destroying face cards, resulting in fewer cards able to trigger Scary Face.

### Blinds

- The Plant ( ) will debuff all face cards, meaning Scary Face Joker cannot trigger.
