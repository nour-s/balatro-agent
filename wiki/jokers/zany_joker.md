# Zany Joker

Copyable Perishable Eternal

Zany Joker is a Common Additive Mult Joker that provides +12 Mult when playing a hand containing a Three of a Kind. This includes hands that include any three cards with the same rank, such as a Full House.

## Synergies

### Decks

- Ghost Deck ( [[Spectral Cards|]] cards may appear in the shop, start with a card Win a run with the on [[Stakes|any difficulty]] ) has several spectral cards (namely Ouija (Converts all cards in hand to a single random [[Card Ranks|]]<br> hand size Available from start. )) that make deck fixing towards Three of a Kind and above much easier.

- Abandoned Deck ( Start run with no [[Card Ranks#Face cards|]] in your deck Win a run with the on [[Stakes|any difficulty]] ) and Mad World ( Extra no longer earn money<br>Earn no at end of round Eternal, Negative <br>Eternal Ranks 2 through 9 only, 32 cards totalBanned Blinds: ) both have smaller deck sizes, making Three of a Kinds much more consistent to find.

- Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ) starts with +2 hand size, making a Three of a Kind easier to search for.

- Erratic Deck ( All [[Card Ranks|]] and [[Suits|]] in deck are randomized Win a run with any [[Decks|deck]] ) can work well with Zany Joker if the seed grants a deck with many cards of the same rank.

### Poker Hands

Playing any hand with three cards of the same rank will trigger Zany Joker. This includes:

- Three of a Kind

- Full House

- Four of a Kind

- Five of a Kind

- Flush House

- Flush Five

### Jokers

- Wily Joker ( ) gives +100 Chips when playing a Three of a Kind.

- The Trio ( if played hand contains a Win a run without playing a . ) gives X3 Mult when playing any hand that has a Three of a Kind, synergizing greatly with Zany Joker's flat +12 Mult.

### Consumables

- The Hanged Man (Destroys up to selected cards Available from start. ) and Immolate (Destroys random cards in hand, gain Available from start. ) can cut deck size down on the size of the player's deck and make assembling a Three of a Kind much easier.

- Death (Select cards, convert the card into the card<br> Available from start. ) can turn cards into other cards and make Three of a Kinds more consistent to put together.

- Ouija (Converts all cards in hand to a single random [[Card Ranks|]]<br> hand size Available from start. ) makes all cards held in hand the same rank at the cost of reducing hand size by -1, making it easier to assemble a Three of a Kind.

## Trivia

Old design of Zany Joker

- This joker is part of a series which provide a Mult boost for a specific hand. The others are Jolly Joker ( ), Mad Joker ( ), Crazy Joker ( ) and Droll Joker ( ).

- In demo versions of the game, Zany Joker had a different color scheme with yellow and red. The Steam page for Balatro still shows this old art in some screenshots.
