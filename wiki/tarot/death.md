# Death

(Drag to rearrange)

Death is a Tarot card that allows the player to convert one card into another by selecting both in a specific order. All properties, such as Enhancements, Editions, and additional chips from Hiker () are copied over.

## Usage notes

- If Death () is used on the automatically-selected card in the Cerulean Bell ( ) Boss Blind, the card converted will also stay permanently selected until played or discarded.

- If Death is used on a debuffed card, the converted card will be debuffed as well.

## Strategies

- Death is a utility Tarot that's mainly used to create extra copies of the player's strongest played cards, or just cards that the player wishes to draw more often. Death is best used to create more Gold Cards ( ) and Steel Cards ( ) as their respective Tarots ( The Devil (Enhances selected card into a [[Card Modifiers|]] Available from start. ) and The Chariot (Enhances selected card into a [[Card Modifiers|]] Available from start. ) respectively) can only enhance one card at a time. Death is also incredibly useful if the player wants to create a copy of a card that has a Seal on it, mainly a card that has a Red Seal () or a Blue Seal (Creates the card for final played [[Poker Hands|]] of round if in hand<br> ).

- A more niche use of Death is using it in a pinch to create a desired poker hand. For example, if the player wants to assemble a Four of a Kind but is down a card needed to complete it, they can use Death to convert one of their unwanted cards into the card they need in order to complete the hand.

## Synergies

### Decks

- Magic Deck ( Start run with the voucher and copies of Win a run with the on [[Stakes|any difficulty]] ) starts off with 2 copies of The Fool (Creates the last [[Tarot Cards|]] or [[Planet Cards|]] card used during this run<br><small> excluded</small> Available from start. ), meaning if found early, the player can create 2 extra copies of Death and give them a major head start in terms of deck fixing.

- Zodiac Deck ( Start run with , , and Win a run with any [[Decks|deck]] ) has Tarot Merchant ( ) making it easier to find copies of Death.

### Poker Hands

- Death is best used to create poker hands that require a lot of deck fixing in order to assemble, especially on a consistent basis. These mainly include the secret hands Five of a Kind, Flush Five and Flush House.

### Jokers

- Fortune Teller ( per [[Tarot Cards|]] card used this run <br> }} ) gains +1 Mult every time Death is used.

### Vouchers

- Tarot Merchant ( ) and Tarot Tycoon ( ) increase the possibility of Death appearing in the Shop.

## Trivia

- In real life, the Death tarot card is often symbolized as rebirth, and does not typically represent actual death, hence its meaning in Balatro.

- Death is one of the two Tarot Cards that appears on the artwork for one of the Jumbo Arcana Packs, the other one is The Emperor (Creates up to random cards<br> Available from start. )
