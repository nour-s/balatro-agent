# The Manacle

StubThis article is a stub. You can help Balatro Wiki by expanding it.

The Manacle is a boss blind that reduces the hand size by 1. For example, if the player's usual hand size is 8, it would be 7 only for this round.

## Strategies

- The Manacle can be less punishing on Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ), as the boss would only reduce the player's hand size to 9 instead of 7, which is still plenty of space to find desired cards (assuming no alterations to hand size were previously made).

- Purchasing the Paint Brush ( ) and Palette ( hand size Reduce your hand size down to five cards. ) Vouchers can help offset The Manacle's penalty.

- Juggler ( hand size Common Effect N/A (Passive) ), Troubadour ( hand size, <br> hand each round Win consecutive rounds by <abbr title="Discards are OK to use.">playing only 1 hand</abbr> Uncommon Effect N/A (Passive) ) and Turtle Bean ( hand size, reduces by each round Uncommon Effect N/A (Passive) ) all give hand size, helping offset The Manacle's penalty.

 - The Juggle Tag ( ) can also be used to counter The Manacle (assuming the tag appears right before the boss blind or the player uses skips).

- Avoid Jokers that reduce hand size such as Stuntman (,<br> hand size Earn at least 100 million (100,000,000) Chips in a single hand. ) and Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ) as the effect will stack with The Manacle.

- Jokers with held in hand abilities vary with The Manacle's effectiveness.

 - Baron (), Shoot the Moon (Each held in hand gives Play every Heart card in your deck in one round. ), and Reserved Parking () are all weakened due to holding fewer face cards in hand.

 - Inversely, Raised Fist () actually benefits, as the player is less likely to hold a card with a low rank at any moment.

- Be mindful of Ectoplasm (Add [[Card Modifiers#Editions|]] to a random hand size Available from start. )'s and Ouija (Converts all cards in hand to a single random [[Card Ranks|]]<br> hand size Available from start. )'s hand size penalty when using them if The Manacle is the current Boss Blind of the Ante.

 - In particular, repeated Ectoplasm usage may make it impossible to form certain Poker Hands feasibly, which is only compounded by the Boss Blind's innate ability.

- The Manacle is able to decrease hand size to 0, resulting in an instant game over, so avoid selecting it with a hand size of 1 if possible.

- It is possible to use the temporary reduction of hand size from the Manacle to unlock Palette ( hand size Reduce your hand size down to five cards. ).

- Luchador (Sell this card to disable the current [[Blinds|]] Uncommon Effect N/A (Passive) ) and Chicot (Disables effect of every [[Blinds|]] Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ) disable the boss blind when their effect is activated, restoring the player's original hand size.

 - Due to how Chicot "disables" boss blinds, having multiple copies of Chicot (Disables effect of every [[Blinds|]] Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ) can allow the player to permanently increase their hand size, with each copy increasing it by 1. This is a more niche strategy of obtaining the elusive naneinf score. However, it is incredibly difficult and expensive to pull off without a lot of set-up.

## Trivia

- The icon for The Manacle is the Phoenician letter Zayin (𐤆), which represents a weapon or manacle in the hieroglyphics from which it is derived.

- A "manacle" is a device used to restrain or confine one's hands (eg. a handcuff or a shackle). This ties into the boss's effect of lowering the player's hand size by 1, which can make it harder to draw certain cards from their deck.

### In other media

- The Manacle makes an appearance as a boss enemy in Vampire Survivors.
