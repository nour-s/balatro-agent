# The Water

StubThis article is a stub. You can help Balatro Wiki by expanding it.

The Water is a boss blind that sets the number of the player's discards to 0 for the round.

## Strategies

- Picking up Burglar (When [[Blind|]] is selected, ) prior to the blind can give additional hands without any downside. The player can use the extra hands as "discards" to find less common poker hands like Straight, Four of a Kind, etc.

- Mystic Summit ( ) will immediately get +15 Mult, since the blind starts with 0 discards.

- Jokers that increase Hand size, such as Juggler ( hand size Common Effect N/A (Passive) ) or Troubadour ( hand size, <br> hand each round Win consecutive rounds by <abbr title="Discards are OK to use.">playing only 1 hand</abbr> Uncommon Effect N/A (Passive) ), can be advantageous for assembling uncommon poker hands. The Paint Brush ( ) and Palette ( hand size Reduce your hand size down to five cards. ) Vouchers can also be useful for this purpose.

 - The Juggle Tag ( ) can also be used to counter The Water (assuming the tag appears right before the boss blind or the player uses skips).

- This blind hard-counters builds focused around discards, such as those focused on Hit the Road (This Joker gains for every discarded this round<br> }} Discard at the same time. ) and Banner ( ).

- It also reduces the amount of money potentially gained from Jokers like Delayed Gratification (Earn per if no discards are used by end of the round Common Economy N/A (Passive) ) and Mail-In Rebate (Earn for each discarded [[Card Ranks|]], ).

- It prevents the player from further building Yorick (This Joker gains every cards discarded<br>}} Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ), Burnt Joker (Upgrade the level of the first poker hand each round Sell 50 cards. ), and Castle (This Joker gains per discarded [[Card Suits|]] card, suit changes every round <br> }} ) stacks.

- Golden Needle ( Discards each cost <br><br> hands per round<br> discards per round<br>Start with Banned Jokers: <br>Banned [[Vouchers]]: , ) is particularly sensitive to this boss blind, given that there can only be one hand each round with no opportunity to fish for new cards.

- Drunkard ( discard each round Common Effect N/A (Passive) ) and Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ) are ineffective against this blind, as their extra discards will not be given.

- Green Joker ( per hand played <br> per discard <br> }} ) and Ramen (, ) cannot lose Mult.

- Luchador (Sell this card to disable the current [[Blinds|]] Uncommon Effect N/A (Passive) ) and Chicot (Disables effect of every [[Blinds|]] Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ) disable the Water's effect, restoring the player's discards.

 - Due to how Chicot (Disables effect of every [[Blinds|]] Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ) "disables" boss blinds, players with multiple copies of the Joker can enter the blind and give themselves more discards than what is normally possible. For example, if the player has 3 discards and has 4 copies of Chicot, they will be given 12 discards upon entering the blind. However, it should be noted that when the Water is defeated and the player moves on to the next ante, their discards will return to their normal amount, making this strategy more of a gimmick than an actual plan.
