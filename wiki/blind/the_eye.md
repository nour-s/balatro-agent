# The Eye

StubThis article is a stub. You can help Balatro Wiki by expanding it.

The Eye is a boss blind that restricts the player from playing a specific poker hand more than once. For example, if the player has triggered a Two Pair in this round, they will not be able to trigger another Two Pair for this round. They will have to resort to using another poker hand, instead. Attempting to trigger a repeated hand type will result in no score being added -- the game says "Not allowed!" and forces Chips and Mult to 0, wasting the hand.

## Strategies

- Card Sharp ( ) is completely nullified by this Boss, since the player will not be able to play the same hand a second time.

- Supernova (Adds the number of times [[Poker Hands|]] ) can have mixed results with this Blind depending on the run so far, as each hand can only be used once. Builds that play many different hands will have an easier time against this Boss than a build that relies on playing a single hand.

- Obelisk (This Joker gains per hand played without playing your most played <br> }} ) can benefit from this boss, since it encourages the player to use different hand types.

- If a run is dependent on only a single hand type, using score-increasing tarot cards such as Justice (Enhances selected card into a [[Card Modifiers|]] Available from start. ) or The Chariot (Enhances selected card into a [[Card Modifiers|]] Available from start. ) may get the score needed in a single played hand. By saving these in the consumable area for this boss, it may save the run.

- It is important to note that Ice Cream (<br> ) and Seltzer () still lose value even if the played hand is a hand that has already been played.

## Trivia

- The icon for The Eye is the Phoenician letter Ayin (𐤏), which represents an eye or mouth in the hieroglyphics from which it is derived.

- LocalThunk has said that The Eye bans repeat hand types because it has observed what hands you've played.1]
