# The Psychic

The Psychic is a Boss Blind that requires the player to play 5 cards in a hand. The hand doesn't require for all of the cards to trigger, but it is required to play 5 cards, otherwise, the player will not be able to score anything at all.

## Strategies

- Using Splash (Every counts in scoring Common Effect N/A (Passive) ) can help with adding extra score when 5-card Poker Hands are unable to be assembled.

- Don't reduce hand size too much, for example with Stuntman (,<br> hand size Earn at least 100 million (100,000,000) Chips in a single hand. ) or Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ). If your hand size is 4 or below, you will be unable to play any 5-card hand.

 - This also applies with Ouija (Converts all cards in hand to a single random [[Card Ranks|]]<br> hand size Available from start. ) and Ectoplasm (Add [[Card Modifiers#Editions|]] to a random hand size Available from start. ).

- DNA (If of round has only card, ), Sixth Sense (If of round is a single , destroy it and create a [[Spectral Cards|]] card <br> Uncommon Effect N/A (Passive) ), and Half Joker ( ) are completely unusable when encountering this boss blind, as they requires hands with less than 5 cards, while The Psychic requires for a hand to always contain 5 cards.

- Square Joker (This Joker gains if played hand has exactly cards <br> }} ) also cannot scale during this Blind.

- It is important to note that Ice Cream (<br> ) and Seltzer () still lose value even if the played hand has 4 or less cards.

## Trivia

- The Icon for The Psychic is the Phoenician letter Kaph (𐤊), which represents palm of a hand in the hieroglyphics from which it is derived.

 - According to LocalThunk, this name comes from palm reading.1]
