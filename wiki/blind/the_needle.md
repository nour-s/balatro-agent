# The Needle

The Needle is a boss blind that sets the player's hands to 1 at the start of the blind.

## Notes

- Getting Troubadour ( hand size, <br> hand each round Win consecutive rounds by <abbr title="Discards are OK to use.">playing only 1 hand</abbr> Uncommon Effect N/A (Passive) ) during The Needle cannot reduce hands per round below 1.

## Strategies

- Burglar (When [[Blind|]] is selected, ) is a great counter to The Needle, as it bypasses the 1-hand restriction and still gives 3 extra hands. This is improved further by copying burglar with Blueprint (Copies ability of [[Jokers|]] ) or Brainstorm (Copies the ability of leftmost [[Jokers|]] Discard a . ).

- Using Luchador (Sell this card to disable the current [[Blinds|]] Uncommon Effect N/A (Passive) ) or Chicot (Disables effect of every [[Blinds|]] Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ) against The Needle gives the player their hands back, but keeps the lowered blind requirement.

- Avoid using Card Sharp ( ), as it becomes completely nullified by this boss (unless the player has Burglar (When [[Blind|]] is selected, )).

- Avoid using DNA (If of round has only card, ) or Sixth Sense (If of round is a single , destroy it and create a [[Spectral Cards|]] card <br> Uncommon Effect N/A (Passive) ), unless the player can score high enough with High Card.

- Using the Justice (Enhances selected card into a [[Card Modifiers|]] Available from start. ) Tarot card to get the X2 Mult may be worth the risk of the card shattering in order to meet the threshold. If possible, it can be saved in the consumable area to ensure it can be used on the single hand given.

- Because the player can no longer use hands as discards, they may struggle to find their ideal poker hand. Some mitigation methods include:

 - A bigger hand size from the Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ), Jokers like Juggler ( hand size Common Effect N/A (Passive) ) and Troubadour ( hand size, <br> hand each round Win consecutive rounds by <abbr title="Discards are OK to use.">playing only 1 hand</abbr> Uncommon Effect N/A (Passive) ), or the Paint Brush ( ) and Palette ( hand size Reduce your hand size down to five cards. ) vouchers.

 - More discards from the Red Deck ( ), Jokers like Drunkard ( discard each round Common Effect N/A (Passive) ) and Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ), or the Wasteful () and Recyclomancy () vouchers.

 - More score form Acrobat ( ) and Dusk ().

## Trivia

- The icon for The Needle is the Phoenician letter Qoph (𐤒), which represents a needle eye in the hieroglyphics from which it is derived.

- The Needle requires precision in reference to the phrase "thread the needle".1]
