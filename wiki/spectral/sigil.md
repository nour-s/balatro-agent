# Sigil

Sigil is a Spectral card that turns all cards held in hand into a single random suit. This effect, while not incredibly impactful, can give the run some direction if used early enough.

## Usage notes

- Sigil cannot be used when hand size is 1.

- Having a large Hand Size makes this card more useful, as more cards will be converted.

## Synergies

### Decks

- Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ) has more hand size, so more cards will be changed at a time.

- Ghost Deck ( [[Spectral Cards|]] cards may appear in the shop, start with a card Win a run with the on [[Stakes|any difficulty]] ) allows the player to hold a Sigil during the round more easily, letting them unlock Flush House earlier.

 - Holding Sigil with Death (Select cards, convert the card into the card<br> Available from start. ) or Strength (Increases rank of up to selected cards by Available from start. ), or when the deck already has more of 1 rank, can allow the player to unlock Flush Five early if the player draws well.

### Poker Hands

- Flush becomes much more consistent if Sigil is used early.

 - If used late, Sigil has anti-synergy because it can turn cards into the wrong suit for what the player is building towards.

- Flush House and Flush Five can be unlocked earlier and played more consistently.

### Jokers

- Castle (This Joker gains per discarded [[Card Suits|]] card, suit changes every round <br> }} ) can be easier to use because it encourages building towards fewer suits, but it does not matter which suits those are.

- Droll Joker ( ), Crafty Joker ( ), and The Tribe ( if played hand contains a Win a run without playing a . ) all become easier to activate.

- Juggler ( hand size Common Effect N/A (Passive) ), Troubadour ( hand size, <br> hand each round Win consecutive rounds by <abbr title="Discards are OK to use.">playing only 1 hand</abbr> Uncommon Effect N/A (Passive) ), and Turtle Bean ( hand size, reduces by each round Uncommon Effect N/A (Passive) ) all allow more cards to be changed at a time.

 - Turtle Bean ( hand size, reduces by each round Uncommon Effect N/A (Passive) )'s downside of limited rounds doesn't matter because Sigil is only used once.

- Greedy Joker (), Bloodstone ( ), and other Jokers dependent on one suit become more powerful if Sigil hits the suit they affect.

 - Sigil has anti-synergy with these Jokers if they are already held, as there is only a 1/4 chance for Sigil to hit the same suit.

### Vouchers

- Paint Brush ( ) and Palette ( hand size Reduce your hand size down to five cards. ) allow more cards to be changed at a time.

### Blinds

- If the player holds Sigil in their consumable slots during The Serpent ( ) and only discards or plays 1 card at a time, they can change the suit of many cards at once.

## Anti-Synergies

### Decks

- Checkered Deck ( ) starts with only  Spades and  Hearts, while Sigil can still make  Clubs and  Diamonds, which defeats the purpose of the deck.

- Five-Card Draw ( discards per round<br> hand size<br> Joker slots <br>Banned Jokers: , , )'s reduced hand size does not allow as many cards to be changed at a time.

### Jokers

- Ancient Joker (Each played card with [[Card Suits|]] gives when scored, <br> <small>suit changes at end of round</small> ) and Flower Pot ( if poker hand contains a card, card, card, ) can be harder to use because of the uneven distribution of suits.

- Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ) and Stuntman (,<br> hand size Earn at least 100 million (100,000,000) Chips in a single hand. ) do not allow as many cards to be changed at a time.

## Trivia

- The word "sigil" refers to ancient, usually primordial symbols that hold mystical powers.

- The symbols found on the card art are the  Heart,  Club,  Diamond, and  Spade suits (in that order).

- Ouija (Converts all cards in hand to a single random [[Card Ranks|]]<br> hand size Available from start. ) has a similar effect, but affects rank instead of suit, and has -1 hand size penalty.
