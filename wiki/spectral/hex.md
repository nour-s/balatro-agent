# Hex

, destroy all other Jokers

Hex is a Spectral card that adds Polychrome ( Edition Joker (Polychrome).png) to one of the currently held Jokers at random, then destroys all other held Jokers.

## Usage notes

- Hex cannot destroy Eternal () Jokers.

- Hex will not apply to any Joker that already has an edition (including Negative ( Joker or Consumable slot Edition Joker (Negative).png)). It also cannot be used if all Jokers already have an edition, and its Use button will be disabled.

- Ghost Deck ( [[Spectral Cards|]] cards may appear in the shop, start with a card Win a run with the on [[Stakes|any difficulty]] ) starts the player with a copy of Hex.

## Synergies

### Decks

- Non-Perishable ( All Jokers are Eternal Banned Jokers: , ; , , ; ; , , and <br>Banned [[Blinds]]: ) works well with Hex, as all Jokers are Eternal (), allowing the player to add Polychrome ( Edition Joker (Polychrome).png) to any Jokers freely without destroying other Jokers.

- Typecast ( When ante boss is defeated:<br>- all Jokers become <br>- set Joker slots to Banned [[Blinds]]: ) also works with Hex, as after defeating Ante 4's boss, all Jokers will become Eternal, allowing the player to add Polychrome to any Jokers freely without destroying other Jokers.

### Stakes

- Playing on Black Stake and above allows Eternal Jokers to appear. Since Eternal Jokers cannot be destroyed, the player can safely use Hex to add Polychrome to any Jokers if they are Eternal.

## Strategy

- Hex grants a random Joker Polychrome ( Edition Joker (Polychrome).png), making it give X1.5 Mult at the expense of destroying all other non- Eternal () Jokers.

- Hex essentially acts as a one-time use, more dangerous, but guaranteed version of The Wheel of Fortune ( chance to add [[Foil|]], [[Holographic|]], or [[Polychrome|]] edition to a random [[Jokers|]] Available from start. ) that always gives Polychrome. On higher stakes, particularly from Black Stake onwards due to the Eternal sticker appearing from this stake, it rises in viability and usage, providing good value for one's run with a downside that may easily be ignored.

- Hex destroys Jokers, which means it's a good idea to sell all the Jokers that you don't want it to grant an Edition, gaining a little bit of money back and leaving a greater chance to hit other Jokers if you have multiple Eternal Jokers.

- If you leave one Joker that is not Eternal, there is always a chance that Hex hits it, which can be helpful for scoring while still keeping the other Jokers that have the Eternal sticker.

 - If all Eternal Jokers have editions (through The Wheel of Fortune ( chance to add [[Foil|]], [[Holographic|]], or [[Polychrome|]] edition to a random [[Jokers|]] Available from start. ) or otherwise) and the non Eternal Joker does not, Hex is guaranteed to hit the non Eternal Joker and is safe to use.

- If your build requires the current Jokers that, while not too beneficial going into the late game, is needed for survival, doesn't function as well with only one Joker that has Polychrome, or suffer greatly if you give a Joker Polychrome and destroy the others, it may not be wise to use Hex, because the effect may not be beneficial to the run and the Joker destruction effect can in turn be run-killing.

## Trivia

- A Hex refers to the bewitching, or casting of spells or curses, on a person, creature, or object.
