# Ectoplasm

hand size

Ectoplasm is a Spectral card that turns a random Joker Negative ( Joker or Consumable slot Edition Joker (Negative).png) at the cost of X hand size, where X = 1 + each time Ectoplasm has been used in the current run. E.g. using Ectoplasm 3 times in the same run decreases hand size by a total of 6 (1+2+3).

## Usage notes

- Ectoplasm picks the base edition Joker that receives the Negative edition randomly, similar to The Wheel of Fortune ( chance to add [[Foil|]], [[Holographic|]], or [[Polychrome|]] edition to a random [[Jokers|]] Available from start. ).

## Strategy

- Ectoplasm grants a random Joker the Negative ( Joker or Consumable slot Edition Joker (Negative).png) making it give +1 Joker slot and essentially frees up a Joker slot. This amazing effect is held back greatly by its -1 extra Hand size for each Ectoplasm used throughout the run, which can be a run killer if not used carefully, especially when considering how valuable Hand sizes are.

- Ectoplasm is oftentimes used to help the run for longer. By having more Joker slots, the player can potentially find new Jokers that can help the run out significantly, or duplicates of strong Jokers already owned when paired with Showman ([[Jokers|]], [[Tarot Cards|]], [[Planet Cards|]], and [[Spectral Cards|]] cards may appear multiple times Reach Ante level Uncommon Effect N/A (Passive) ).

- Ectoplasm can only target Jokers with the Base (No extra effects Edition Joker (Base).png) Edition, which means it is possible to force the hand of fate and make it hit the Joker that is wanted easily.

 - This is done by selling all Jokers with the Base edition that the player doesn't want Ectoplasm to hit, of course leaving the desired Joker as the only Joker left (besides Jokers that already have an Edition). Note that this action may kill the run if the Jokers sold are essential to the run's scoring.

 - Note that this also means Ectoplasm can't be used if all Jokers that the player possesses already has an Edition.

- If Ectoplasm take effect upon a debuffed Joker ( Perishable () Joker after five rounds), the Joker slot will not increase, and selling it will result in permanent Joker slot loss, which is likely to be a bug.

- Due to the heavy penalty of using Ectoplasm, it is recommended the player first obtain more Hand sizes, be it granted by the Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ), permanent by the Paint Brush ( ) and Palette ( hand size Reduce your hand size down to five cards. ) vouchers, or replaceable by Juggler ( hand size Common Effect N/A (Passive) ) or Troubadour ( hand size, <br> hand each round Win consecutive rounds by <abbr title="Discards are OK to use.">playing only 1 hand</abbr> Uncommon Effect N/A (Passive) ).

 - The Hand size deduction effect of Ectoplasm can exceed the negatives, but will not end the run until the next Blind, when it sets the Hand size to 0. This can be taken advantage of by using Perkeo (Creates a [[Negative|]] copy of random [[Consumables|]] card in your possession at the end of the [[The Shop|]] Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. Legendary Effect N/A (Passive) ) to make multiple copies of both Ankh (Create a copy of a random [[Jokers|]], destroy all other Jokers <br> from copy)}} Available from start. ) and Ectoplasm and use them on one or more Eternal () Jokers, with the most common one being Joker Stencil ( for each empty [[Jokers|]] slot <br> <small>Joker Stencil included</small> <br> }} ), to gain a high score for one single round.

 - Additionally, since Balatro only sets the Hand size to 0 afterwards, any methods of gaining Hand sizes in the next shop (most reliably the Paint Brush and Palette vouchers, or selling Jokers that deduct Hand size such as Stuntman (,<br> hand size Earn at least 100 million (100,000,000) Chips in a single hand. ) or Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) )) allows continuing the run as normal, provided the player can defeat the next blinds with such limited Hand size.

- A major warning for using multiple Ectoplasms is that, for example, using 3 of them gives -6 Hand sizes, which on most runs means only being able to hold 4 cards or less at a time, making The Psychic ( ) a dead-end if no methods of disabling or rerolling the Boss blind is available, like Luchador (Sell this card to disable the current [[Blinds|]] Uncommon Effect N/A (Passive) ), Chicot (Disables effect of every [[Blinds|]] Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ), Director's Cut (Reroll [[Blinds|Boss Blind]] time per Ante, per roll Base Director') or Retcon (Reroll [[Blinds|Boss Blind]] times, ), or ways to hold more cards like Certificate (When round begins, add a random with a random [[Seal|]] to your hand Have a [[Gold_Cards|Gold playing card]] with a [[Gold_Seal|]] Uncommon Effect N/A (Passive) ) or Cryptid (Create copies of selected card in your hand Available from start. ).

- If the build requires the current Jokers that, while not too beneficial going into the late game, is needed for survival, doesn't need more Jokers to function, or suffer greatly from not having many Hand sizes, it may not be wise to use Ectoplasm, because the effect may not be beneficial to the run and the Hand size deduction effect can in turn be run-killing.

## Trivia

- "Ectoplasm" is known as a supernatural substance that is "exteriorized" by physical mediums. In popular media, it is generally shown to be a viscous substance that is created by ghosts and spirits.

- The image features candles because candles are a common set piece for summoning circles, fortune telling, Ouija reading, and other attempts to contact the paranormal.

- Ectoplasm appears in the artwork for the Mega Spectral Pack.
