# Ankh

from copy)

Ankh is a Spectral card that picks a random Joker, and destroys all other Jokers. It then creates a copy of that randomly chosen Joker.

## Usage notes

- The copied Joker will retain almost all the properties of the original, including any scaling, Edition, and Stickers. There is only 1 exception: Negative ( Joker or Consumable slot Edition Joker (Negative).png) is removed from the copy.

- Ankh cannot destroy Eternal () Jokers, though this does not impact which Joker can be chosen as target.

- Ankh cannot be used if the player's Joker slots are all filled, it will show a "No Space!" message.

- It is generally recommended to sell all other Jokers except the target Joker.

- Copying a Joker does not copy the card back - the copied Joker will always be created with the Red Deck ( )'s card back and can thus be identified even when turned face down by the Amber Acorn ( ) boss blind. This is presumed to be a bug.

## Synergies

### Decks

- Non-Perishable ( All Jokers are Eternal Banned Jokers: , ; , , ; ; , , and <br>Banned [[Blinds]]: ) works well with Ankh, as all Jokers are Eternal (), allowing the player to duplicate any Jokers freely without destroying other Jokers. However, it becomes useless once all Jokers slots are filled.

### Stakes

- Playing on Black Stake and above allows Eternal Jokers to appear. Since Eternal Jokers cannot be destroyed, the player can safely use Ankh to duplicate any Jokers if they are Eternal.

## Strategy

- Ankh creates a duplicate of random Joker at the expense of all non- Eternal () Jokers, giving the player 2 copies of a Joker, essentially doubling its value. This effect has lots of potential to help the run pivot in multiple different ways depending on the Joker it chooses to copy, but be careful, it can mess up the run as well if not considered carefully.

- Ankh essentially acts as a one-time use, more dangerous version of Invisible Joker (After rounds, sell this card to a random Joker <br> /2}} <br> ]] from copy)}} Win a run without ever having more than Rare Effect N/A (Passive) ). On higher stakes, particularly from Black Stake onward due to the Eternal sticker appearing from this Stake, it pivots in liability and usage, providing immense value for one's run with a downside that may easily be ignored.

 - Note that Ankh also copies the Perishable () and Rental () stickers, which can be hugely detrimental.

- Ankh destroys Jokers, which means it's a good idea to sell all the Jokers that you don't want it to copy, gaining a little bit of money back and leaving a greater chance to hit other Jokers if you have multiple Eternal Jokers and only one of them you want to duplicate.

- If you leave one Joker that is not Eternal, there is always a chance that Ankh hits it, doubling that Joker which can be helpful for scoring while still keeping the other Jokers that have the Eternal sticker.

- If your build requires the current Jokers that, while not too beneficial going into the late game, is needed for survival, doesn't function as well with two Jokers of the same type, or suffer greatly if you duplicate a certain Joker and destroy the others, it may not be wise to use Ankh, because the effect may not be beneficial to the run and the Joker destruction effect can in turn be run-killing.

## Trivia

- An "Ankh" (☥) is a symbol of life in ancient Egyptian culture and is said to be the original cross.

- The line "(Removes Negative from copy)" in the card's effect only shows if you already have a Negative ( Joker or Consumable slot Edition Joker (Negative).png) Joker.

- If the player tries to buy and use Ankh in the shop (only on Ghost Deck ( [[Spectral Cards|]] cards may appear in the shop, start with a card Win a run with the on [[Stakes|any difficulty]] )) while all Joker slots are filled, the Ankh card will become stuck in place for the rest of the run. The player can actually interact and move the bugged Ankh around. This is likely a bug.

 - Strangely, this bug doesn't happen if the player uses Ankh in the consumable slot or in the Booster Packs.

- When Ankh duplicates a Joker, the color of spawning animation changes with the rarity of Joker. For example, if a Rare Joker is duplicated, the animation will be red. Ankh is the only item with this property, as most items do not have a spawning animation (such as Judgement (Creates a random [[Jokers|]] card<br> Available from start. )), or can only generate a specific rarity (such as Riff-Raff (When [[Blind|]] is selected, create [[Common Jokers| ]] <br> )).
