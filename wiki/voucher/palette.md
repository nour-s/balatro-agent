# Palette

( )

Palette is a voucher that increases hand size by 1.

## Usage notes

- The player must reduce their hand size to 5 cards in a run to unlock Palette.

 - This can be achieved through a combination of Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ), Stuntman (,<br> hand size Earn at least 100 million (100,000,000) Chips in a single hand. ), Ectoplasm (Add [[Card Modifiers#Editions|]] to a random hand size Available from start. ), Ouija (Converts all cards in hand to a single random [[Card Ranks|]]<br> hand size Available from start. ), and/or The Manacle ( ).

- Paint Brush ( ) must be purchased in the same run for Palette to appear.

## Synergies

### Decks

- Luxury Tax ( Hold cards in hand for every you have<br><br> hand size ): Palette can help offset the hand size penalty given by this deck.

- Five-Card Draw ( discards per round<br> hand size<br> Joker slots <br>Banned Jokers: , , ): Palette can partially offset the deck's -3 hand size penalty.

### Jokers

- Any Joker that benefits from held in hand procs will appreciate the extra hand size.

 - Having a bigger hand size will allow you to hold more Kings for Baron (), and more Queens for Shoot the Moon (Each held in hand gives Play every Heart card in your deck in one round. ).

 - Reserved Parking () can proc more if you're able to hold more face cards in hand, allowing you to earn more money.

 - Mime () can retrigger more cards this way as well. This can allow you to retrigger Blue Seals (Creates the card for final played [[Poker Hands|]] of round if in hand<br> ), Gold Cards ( ), Steel Cards ( ), or the Jokers mentioned above.

- This also benefits Sixth Sense (If of round is a single , destroy it and create a [[Spectral Cards|]] card <br> Uncommon Effect N/A (Passive) ) by making it much easier to find your 6s. However this is not a long term synergy, unless you combine it with DNA (If of round has only card, ), which makes the combination a lot easier, as you are more likely to find what you need.

- Palette can help offset Jokers that reduce hand size such as Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ) and Stuntman (,<br> hand size Earn at least 100 million (100,000,000) Chips in a single hand. ).

### Poker Hands

- Poker hands that require 5 cards to play like Full House, or Straight can benefit from the large hand size, making it easier to find the cards to make them.

- Smaller hands like High Card or Pair also benefit from large hand sizes, by allowing you to hold onto more Steel Cards ( ) to increase your score.

### Consumables

- Palette can help offset Spectral Cards that reduces hand size such as Ouija (Converts all cards in hand to a single random [[Card Ranks|]]<br> hand size Available from start. ) and Ectoplasm (Add [[Card Modifiers#Editions|]] to a random hand size Available from start. ).

### Modifiers

- This also makes it easier/faster to draw Purple Seal (Creates a card when <br> ) cards.

- The increased hand size can allow for more Gold Cards ( ), Steel Cards ( ), and Blue Seal (Creates the card for final played [[Poker Hands|]] of round if in hand<br> ) to be utilized.

## Trivia

- Painted Deck ( hand size, <br> Joker slot Win a run with any [[Decks|deck]] ), Paint Brush ( ), and Palette () share the theme of painting and increased hand size.

 - This might be because the player "draws" more cards with a higher hand size.

 - This may also represent how painting often involves multiple colors, and the cards in Balatro come in a variety of colors (especially when involving Card Modifiers), the Painted Deck and the paint vouchers increasing hand size allows more cards, and by extension, more colors, to be on the screen at once.
