# Director's Cut

per roll (Reroll [[Blinds|Boss Blind]] times, )

Director's Cut is a voucher that allows the player to reroll the boss blind once each Ante for $10.

## Usage notes

- Rerolling on Ante 8 (or a multiple of 8) will always roll a Showdown Blind, and other antes will always roll a normal Boss Blind.

## Synergies

### Decks

- Green Deck ( At end of each Round: <br> per remaining <br> per remaining <br> Earn no Discover at least items from your [[collection]] ) can allow you to gain more money in order to purchase then use this voucher.

- Yellow Deck ( Start with extra Discover at least items from your [[Collection|collection]] ) may help you with the cost of this as well.

### Poker Hands

Any poker hands you have that are countered by the current boss blind can be avoided, by using this voucher. For example, The Plant ( ) can be changed into a boss that does not debuff a deck reliant on face cards.

### Jokers

- Matador () can benefit depending on what the rerolled boss blind becomes, and your own deck.

- Having a strong economy with Jokers like To the Moon (Earn an extra of for every you have at end of round Uncommon Economy N/A (Passive) ), Satellite (Earn at end of round per unique card used this run Have at least . Uncommon Economy N/A (Passive) ), Mail-In Rebate (Earn for each discarded [[Card Ranks|]], ), etc. may be necessary to keep rerolling the Blinds since it's expensive.

## Anti-Synergies

### Jokers

- Chicot (Disables effect of every [[Blinds|]] Find this [[Jokers|Joker]] from the [[Spectral Cards|]] card. ) and Luchador (Sell this card to disable the current [[Blinds|]] Uncommon Effect N/A (Passive) ) can disable the current boss effect, making this voucher less useful.

 - However, since Chicot’s ability to disable boss blinds allows you to gain Hand size if you have multiple of that card when facing The Manacle ( ), it can be worth it to reroll for the Manacle. Unfortunately, due to this requiring you to have multiple Chicots, it is unlikely that this strategy can be viable.

### Tags

- When using the Boss Tag ( ), the game will also consider the reroll from the voucher to be used. This means that the player will not be able to reroll the current ante's boss blind again using Director's Cut after taking this tag.

## Trivia

- Director's Cuts are the version of the film the director personally envisioned. This voucher is similarly, changing the original version of the run, into something the player would prefer.
