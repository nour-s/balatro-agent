# Overstock Plus

( )

Overstock Plus is a voucher that increases the number of card slots available in the shop from 3 to 4. It also immediately restocks any empty card slots in the Shop when purchased.

## Usage notes

- The player must spend a total of $2500 at shop across runs to unlock Overstock Plus.

- Overstock ( ) must be purchased in the same run for Overstock Plus to appear.

## Synergies

- Chaos the Clown ( free per shop Common Effect N/A (Passive) ), Reroll Surplus () and Reroll Glut () can make rerolling very lucrative and allow the player to get even more value for less money.

- Flash Card (This Joker gains per in [[The Shop|the shop]] <br> }} ) has a similar want of rerolls, and having more shop slots naturally makes rerolling in general more worth it.

 - One can also consider that more Shop slots means more chances for items that may provide money like The Hermit (Doubles money<br>)}} Available from start. ), which then means more possible rerolls for Flash Card (This Joker gains per in [[The Shop|the shop]] <br> }} ).

- Having more shop slots allows Jokers such as Campfire (This Joker gains for each card , resets when [[Blinds|]] is defeated <br> }} ), Constellation (This Joker gains every time a [[Planet_Cards|]] card is used <br> }} ), and Fortune Teller ( per [[Tarot Cards|]] card used this run <br> }} ) to scale faster.
