# Petroglyph

( Ante, <br> )

Petroglyph is a voucher that decreases the ante by 1, at the cost of 1 discard.

## Usage Notes

- The player must reach Ante 12 in a run to unlock Petroglyph.

- Hieroglyph ( Ante, <br> ) must be purchased in the same run for Petroglyph to appear.

- Petroglyph can cause certain Boss Blinds or Showdown Blinds to appear on antes they wouldn't normally be able to appear on, since the Boss Blind doesn't change after Petroglyph is purchased. Rerolling the Blind after purchasing Petroglyph will change it back into an expected Boss Blind.

- Ante 0 can be reached by purchasing a Petroglyph on Ante 1.

 - Ante -1 can be reached via incredibly rare circumstances, involving a voucher tag, very fast reflexes, and the seed to roll both Hieroglyph and Petroglyph. A basic demonstration can be found here.

- Petroglyph actually just reduces the Ante value by 1, without ending or starting Antes, so Ante-related effects (such as The Pillar ( )) will still work as usual.

- If the player obtains Petroglyph in Ante 9, returns to Ante 8 and fails, non-Endless mode game over scenes will be used (i.e., the background turns red and Jimbo appears). This may be a bug.

## Synergies

### Decks

- Red Deck ( ) gives +1 discard, which can be helpful by mitigating the -1 discard penalty from Petroglyph.

### Jokers

- Jokers that rely on scaling can benefit from Petroglyph with the extra Ante. By playing more rounds, you give your Jokers more chances to get stronger.

 - For example: Square Joker (This Joker gains if played hand has exactly cards <br> }} ), Lucky Cat (This Joker gains every time a [[Lucky Cards|]] card triggers <br> }} ), Castle (This Joker gains per discarded [[Card Suits|]] card, suit changes every round <br> }} ), etc.

 - Alternatively, Throwback ( for each [[Blinds|]] skipped this run <br> }} ) can gain more XMult by skipping more Blinds.

- Jokers that can potentially reward you for playing more Blinds. You can get more value from these Jokers this way.

 - Economy Jokers such as Satellite (Earn at end of round per unique card used this run Have at least . Uncommon Economy N/A (Passive) ), Golden Joker (Earn at end of round Common Economy N/A (Passive) ), Mail-In Rebate (Earn for each discarded [[Card Ranks|]], ), etc.

 - Tarot and Spectral Card Jokers such as Vagabond (Create a [[Tarot Cards|]] ), Superposition (Create a [[Tarot Cards|]] card if poker hand contains an and a <br> ), Séance (If [[Poker Hands|]] is a , create a random [[Spectral_Cards|]] card <br> ), etc.

- With up to three more Blinds with the additional Ante, Burnt Joker (Upgrade the level of the first poker hand each round Sell 50 cards. ) will get more chances to activate.

 - This also includes Jokers like Trading Card (If of round has only card, ), Marble Joker (Adds one [[Stone Cards|]] card to the deck when [[Blinds|]] ), Space Joker ( chance to upgrade level of played [[Poker Hands|]] ), etc.

- Cards with Seals can be activated more with the extra Blind.

- Drunkard ( discard each round Common Effect N/A (Passive) ) can provide +1 discard, which can be helpful by mitigating the -1 discard penalty from Petroglyph.

- Merry Andy ( discards each round, <br> hand size Win a run in or fewer rounds Uncommon Effect N/A (Passive) ) can provide +3 discards, mitigating the -1 discard penalty from Petroglyph at the cost of -1 hand size.

- Burglar (When [[Blind|]] is selected, ) removes all discards, making Petroglyph's -1 discard penalty irrelevant.

## Anti-Synergies

### Decks

- Green Deck ( At end of each Round: <br> per remaining <br> per remaining <br> Earn no Discover at least items from your [[collection]] ) will give the player $1 less at the end of each blind.

### Jokers

- Jokers that require discards, such as Banner ( ), Trading Card (If of round has only card, ), Mail-In Rebate (Earn for each discarded [[Card Ranks|]], ), etc. will be weakened due to fewer discards.

### Tags

- Garbage Tag ( ) will give less money.

### Stakes

- On Blue Stake or higher, Petroglyph becomes more difficult to use with the -1 discard restriction from the start of the run. This means it may be more difficult to get the poker hand you need before the end of the round.

## Trivia

- A petroglyph is a rock carving, especially a prehistoric one. Due to petroglyphs being a prehistoric form of art, the effect of the voucher sending the player back another ante can be interpreted as them being sent further backwards in time.
