# Antimatter

( )

Antimatter is a voucher that adds one additional Joker slot, making it arguably the best Voucher in the game.

## Usage notes

- The player must redeem Blank ( ) 10 times in total during unseeded, non-challenge runs to unlock Antimatter.

- Like other upgraded vouchers, the base voucher ( Blank ( )) must be purchased in the same run for Antimatter to appear.
